# 北极星愿景 — 中国史 4X · 内生反滚雪球

> 2026-07-12 | 第四个 AI 与用户共同起草 | **活文档,非合同**
> 记录最长远的可能状态 + 分析思考。愿景会随认知演进而变,本文保持更新。
> 上游真相仍是 `CLAUDE.md` 顶部;本文是"我们要往哪去 + 为什么"的完整记录。

---

## 0. 一句话

**做一个基于中国历史、六边形地图、小百到中百回合的 4X 策略游戏,用一个内生的反滚雪球机制(现代红白两线分叉,选哪条取决于你自己的发育状态)去解决文明六"滚雪球+平推"的病;以人类玩家作为深度的评判标准和 AI 的训练数据源。**

---

## 1. 核心设计论点(骨中之骨)

> **玩家的发育状态本身,应当成为决策点。**

这是整个愿景的心脏,也是它区别于文明六的根本。展开:

- 文明六是**滚雪球游戏**:领先者选择更多、越滚越大;落后者只能挨打。三星法坦 = 克隆人罗马军团 = 本质都是"造更强的军队,把握时机平推对面"。时代之间只有**数值提升 + 操作提升,没有思路的质变**。
- 各种补丁(黑暗时代/紧急事件/忠诚度)都是**外部橡皮筋**——生硬,玩家能感觉到系统在硬拉。
- **我们的解:让"处境"决定"最优路线",而不是让"领先"单调地决定"更强"。**
  - 滚起来了 → 走能兑现领先的路线,但要付负面代价。
  - 没滚起来 → 走有非线性追赶机制的路线,有特殊的翻盘窗口。
  - **领先玩家和落后玩家面对的最优策略不一样 → 不对称 → 不存在统一支配的"滚雪球+平推"解。**

这不是施舍式的橡皮筋,是**带取舍的选择**。这正是"选择有差异化回报"——策略深度的定义。

## 2. 完整愿景(最长远状态)

### 2.1 基本形态
- 六边形地图(继承当前引擎的轴向坐标+环面,规模需放大)。
- **迷雾(FOW)**:信息博弈,侦察终于有价值。← 深度骨。
- 回合数:小百到中百(~100-300)决出胜负。
- 背景:中国历史。后发优势——文明系列对中国的挖掘其实很浅,而中国本身有深厚的历史素材。

### 2.2 四时代
| 时代 | 主题 | 核心玩法 |
|------|------|---------|
| **远古** | 探路、初步发育 | 迷雾探索、找资源、起步(有迷雾,侦察有用) |
| **封建** | 领袖特色发育 | 各领袖特色展开,拉开/追赶发育差距 |
| **现代** | **红白分叉(分水岭)** | 资本主义(白)vs 社会主义(红),各有特色机制 ← 深度核心 |
| **未来** | 落实胜利 | 各线兑现各自的胜利方式(科技/共产主义/文化/统一…待定) |

**设计意图**:远古/封建拉开发育差距**但不可逆到无所谓**(留悬念到现代分叉);现代分叉让处境变成决策;未来兑现。**漫长的游戏过程不是简单滚雪球,而是发育情况本身持续成为决策点。**

### 2.3 现代红白两线(机制草案 — 需 Phase 1.5 验证,数值全待定)

> ⚠️ 这是最关键、也最难平衡的部分。以下是**设计意图草案**,不是定稿。
> ⚠️⚠️ **已被 §8.5 修正**:下面的"产出乘子"框架太浅(正向等价无张力);
> 正确框架是"负面维度(支持度)+ 红白=两种取舍管理方式",见 §8.5。本节保留作思路演进记录。

**白线(资本主义 / 先发兑现):**
- 强项:资源攫取效率高(市场/贸易/私有资本加速)、扩张快、领先能继续滚。
- **负面机制(平衡的命门)**:内部不稳定(分化→支持度/稳定度下降)、周期性危机(产出骤降)、对外依赖。让先发优势**不是纯赚**,是"高回报高波动"。
- 适合:已领先、想快速终结的玩家。

**红线(社会主义 / 后发追赶):**
- 强项:计划动员(集中资源爆发式投单一方向)、**追赶机制(如"西南联大"=一次性动员换科技弯道超车)**、稳定度高(抗危机抗灾)、集中力量办大事。
- 成本:前期总产出可能低于白线全速、动员有机会成本、依赖正确的时机/路线判断。
- 适合:没滚起来、需要非线性追赶的玩家。
- **红线内部再分先后手**(苏联式重工优先 vs 中国式渐进改革)——**二阶深度,先不做,验证一层分叉有深度再谈嵌套**。

**历史母题(设计动机,不是说教)**:中国近现代不是只有一条路,而是无数仁人志士几乎把每条路线都试了一遍,实践证明中特社最符合实情。**游戏里我们可以模拟不同历史/社会条件下的中国,在现代走出不一样的路**——这天然就是"处境决定路线"的叙事载体。

**关键设计原则**:分叉的回报**取决于你的状态**,所以"领先→白"和"落后→红"都可行,制造出杀死单一支配线的不对称;且每条线对另一条有反制(白线的不稳定可被利用,红线的慢前期可被压制)。

### 2.4 胜利条件(未来时代,待定)
- 各线落实各自胜利:科技胜利、共产主义/大同、文化、经济、军事统一……
- 因为红白已分流,**不同玩家以不同形态抵达未来** → 胜利路径本身也分化。
- ⚠️ 这块用户明确"还没想好",留 OPEN。

### 2.5 对文明六被诟病点的针对性改进(结合中国实际)
| 文明六的病 | 我们的改进 | 骨/皮 |
|-----------|-----------|-------|
| 沙漠/海洋是没什么可开发的死地形 | 植树造林(沙漠→可用)、填海造陆(海→陆)、移动渔场 | 皮(除非开发选择互相抢有限资源/时间=取舍→才是骨) |
| 灾难无新意、收益少不稳定、相关奇观没人造 | 军队可抗震救灾(提升支持度)、大禹式治水领袖 | **小骨**:军队变"打仗 vs 救灾"的可争用资源(机会成本) |

## 3. 骨 vs 皮 分类(砍 scope 时的救命清单)

> 骨 = 产生 AI 可搜索的策略张力;皮 = 人类好玩的丰富度,但对 AI 是额外维度(敌人)。

**深度骨(AI 友好,优先):**
- ✅ **红白不对称分叉(处境决定最优路线)** ← 唯一的核心宝石
- ✅ 迷雾 FOW(信息博弈,复活侦察)
- ✅ 发育 vs 军事节奏取舍(继承当前引擎的建设↔军事张力)
- ✅ 军队打仗 vs 救灾的机会成本(小骨)
- ⏳ 红线内部先后手嵌套(二阶,验证一层后再说)

**人类好玩皮(AI 不友好,后置):**
- 🎨 几十个领袖(全愿景;每个=AI 要学的额外维度)
- 🎨 丰富地形开发(植树/填海/渔场,除非有取舍)
- 🎨 历史事件/奇观/美术叙事
- 🎨 UI/UX/上手引导/手感

**判据**:砍 scope 时,骨全留,皮按"人类反馈需要多少"渐进加。**别让皮淹没骨。**

## 4. 反滚雪球机制的诚实批评(尊重它,才 stress-test 它)

不对称机制是游戏平衡里最难的东西,Civ 挣扎了几十年。三个真风险:

1. **变态激励**:红线追赶太强 → "故意前期摆烂等红线"成最优 → 比滚雪球更糟的退化。刀刃很窄。
2. **平衡刀尖**:红线太强→大家躺平走红线;太弱→滚雪球照赢→回到文明六。**找这个刀尖恰恰需要 AI+裁判工具**——这说明方法论不是负担,是这个想法能落地的前提。
3. **红线内部嵌套**:做好是二阶深度,做不好是复杂度噪声。先证一层。

## 5. 深度规格(可证伪的张力清单 — 消除"不知道深度标准"的忐忑)

> 别把深度当抽象标量去测量。对每条想要的张力,预先写"怎么算它成立",让 `bin/judge` / 人类对局去验。
> **深度 = 已实现的规格数量。**

**规格1(反滚雪球核心)**:封建末领先的玩家选白线胜率应显著>选红线;落后的玩家选红线胜率应显著>选白线。**若两种处境都选同一条线 → 分叉是假的。**

**规格2(红线不是反向滚雪球)**:"前期摆烂→红线暴发"策略的胜率必须<正常发育。否则催生变态激励。

**规格3(现代前不能已决出)**:若 >80% 对局在进入现代前胜负已定 → 分叉来得太晚,是装饰。远古/封建差距要"可追但不可逆到无所谓"。

**规格4(每条线有反制)**:白线的不稳定期应能被对手利用;红线的慢前期应能被对手压制。若某条线无论对手怎么打都稳赢 → 支配,失败。

**规格5(信息有价值)**:开迷雾后,"侦察→预判对手路线→提前反制"的玩法胜率应>无视侦察。否则迷雾/侦察仍是死机制。

---

## 6. 深度的经验主义定义(本项目最重要的认知转变)

> **一个游戏有深度 = 存在一条"新手→高手"的长技能梯度,高手稳定赢新手,且大量对局后仍无已知支配解。**

- 这比"数学证明最优解不存在"弱,但**可验证**:Elo 分布、技能曲线、"没人找到必胜法"。
- 文明六最优解没被数学证明,但玩家实践证明能玩出成果——**人类实践本身就是深度的存在性证明**。
- **这反转了项目一直撞的墙**:不必"先有全知 AI 当裁判"(我们造不出),改成"**让人玩→人验证深度、人产生数据→数据喂 AI**"。AlphaGo 也是先吃人类棋谱。
- **裁判从"一个全知 AI"变成"一群在学习的人类"。** 这是从 S2 循环论证里彻底走出来的路。

| **P2** | 平衡点游戏:红白核心+迷雾+极简时代+3-5领袖+够用UI。做**完整、值得完成**、能给人玩 | 可发布成果+博客+玩家+人类数据 | scope 是新技能(UI/手感);顺序=可玩切片→人→数据→AI;收益社区当上行不当地基 |

**P2 平衡点的最小完整定义**:红白核心分叉 + 迷雾 + 极简三/四时代 + 3-5 领袖 + 够用 UI 让人上手。**够验证深度、够给人玩、够 AI 能搜——不多一分。**

**顺序翻转(重要)**:不是"做完→训 AI→给人玩",而是"**做出可玩切片→先让几个人玩→人类数据定评判标准+喂 AI**"。人先于强 AI 到场,因为人是数据源。别等 AI 训好才开门。

## 8.5 重大修正一:正向等价没有张力 → 负面维度(支持度)

> 2026-07-12 讨论追加。**这条修正了第 2.3 节的红白乘子草案(已作废)。**

**用户洞察**:如果资源/科技/兵力全是正向、单调、可互换,游戏就坍缩成**一个标量**(战力=f(资源,科技,兵力)),最优解=梯度上升="把战力最大化"——**没有任何维度你必须牺牲才能换另一个,那不是决策,是执行**。

**这解释了 S2 为什么裁出"浅"**:当前游戏只有一根轴(发育→战力→赢),没取舍。红白线若只是"不同的正向乘子",state-aware 会退化成"谁乘子大选谁"=算术,不是深度。

**修法:引入正交的负面维度制造取舍。** P1.5 用 **支持度/稳定度(support)**:
- 拉高战力要扣支持度(穷兵黩武/野蛮扩张/占领 → 支持度降)。
- 支持度低有惩罚(产出打折 → 极低触发动荡:掉兵/掉城/掉资源)。这是**内生负反馈**,不是外部橡皮筋。
- "造多少兵/扩多快"第一次成为真决策。

**红白线 = 两种"管理这个取舍"的活法**(不是"哪个乘子大"):
- **白线(资本先发)**:猛拉原始资源,用资本**暂时压制**支持度惩罚(花钱买稳定)→ 高回报高波动(压制周期性失效=危机)。
- **红线(社会后发)**:承认资源 race 追不上,**优先拉支持度/组织度**,再用机制(西南联大/集中力量)**兑换成**科技/产能爆发。赢在正交轴再兑现。

**张力 = 选在哪根轴竞争 + 怎么跨轴兑换。** 深度核心从"验红白乘子"改成"验负面维度+跨轴兑换能否造张力"。

## 8.6 重大修正二:多人·协作·平台分组轴

> 2026-07-12 讨论追加。**"多人"不是一件事,是两种不同深度结构。**

**FFA(每人为己)→ 政治/频率依赖深度**:渔翁得利、集火领先者、临时结盟、背叛。红白频率依赖(全白撞车→红渔利;全红→白一波拍死)活在这里。**脏毛病:kingmaking**(快输的人决定谁赢)→ 对 AI 训练是噪声。

**队伍(2 队各 k 人)→ 协作/分工深度**:用户"rush+发育组队 > 两个不会合作的强 AI"的直觉在这。胜负干净(队赢/队输)、kingmaking 噪声小 → **对 AI 训练和深度测量更友好**。

**"菜队 > 强单干"是把双刃探针**:赢 → 协作是深度源**且**我们 AI 太弱找不到它(又是那个主题);强 AI 本该自己学会协作。**警惕**:涌现协作(学通信)= MARL 前沿(参见 Cicero);脚本/隐式协作(固定角色读棋盘)= 可做。P1.5 只做后者。

**关键统一(用户直觉推到底)**:协同内容(互补领袖/兵种/红白角色)**模式无关**,两种模式共享;但结构差异(盟可流动+可背叛+个人胜利 vs 固定+共享)**不被内容抹平**。
> **队伍 = "预先锁定结盟"的 FFA。两者是"结盟承诺度"连续轴上的两点。**
> → **平台决策:玩家分组做成配置轴(1v1/2v2/4v4/N-FFA/自由结盟),不硬编码模式。** 协同内容模式无关 → 2v2 验出的协同深度顺轴迁移到 FFA。这就是"综合平台"的本质。

**EGTA(经验博弈论分析)= 多人可验证裁判**:准备策略池,跑混合阵容多局,量每策略在不同场上分布下的排名 → 看频率依赖("没有策略是不管别人怎么选都最优"=深度)。**不需要一个通盘强 AI,只要策略池+锦标赛统计** → 部分绕开"造不出强 AI"死结。**1v1 的深度要强 AI 当裁判(造不出);多人深度可用 EGTA 策略池当裁判(造得出)。**

## 8.7 元规律 + 设计轴选项库(记全,待 committed 选择)

**元规律(同一病理三次出现)——"底料太薄 → 假阴性":**
- 单标量 → 要负面维度(否则无取舍)
- 单城 → 要空间(否则扩张取舍无牙)
- 1v1 → 要多人(否则红白频率依赖/协作展不开)
> **深度不在任何单一机制,在"有没有足够正交维度让策略互动涌现"。** 但**这和"保持简单以便可验证"正面冲突**——这是项目核心张力。解法:**一次加一个维度,每加一个带 instrumentation 回答:(a)必需吗(没它张力不被行使)?(b)加了还验得动吗?**

**设计轴选项库(全部发散过的维度,冻结待选):**
| 轴 | 选项 | P1.5 committed |
|----|------|----------------|
| 战力维度 | 单标量 / +负面维度(支持度) | **+支持度** |
| 空间 | 单城 / 抽象扩张 / 多城开拓者 | 先单城抽象扩张,数据不够升多城 |
| 玩家数 | 1v1 / 2v2 / 4v4 / N-FFA | **1v1 烟雾 → 2v2 深度** |
| 分组结构 | 硬编码模式 / **配置轴** | **配置轴(先开1v1+2v2)** |
| 协作 | 无 / 脚本隐式 / 涌现通信(MARL) | **脚本隐式** |
| 阵容 | 固定 / draft | 先固定,后 draft |
| 迷雾 | 无 / FOW | **无(留 P2)** |
| 时代/领袖 | 无 / 极简 / 几十 | **无(P1.5 不做)** |
| 评估 | 2人minimax / 队vs队 / EGTA | 队vs队 + 小 EGTA |

> **纪律:设计可无限发散,验证必须收敛到一个具体形态。不 build 任何东西的完美设计价值为零。**



- **北极星转向确认**:原决策#1"AI友好>人类好玩、做研究平台"。本愿景实质把北极星扩成"人机共玩、以人类为深度基准、反哺 AI 的完整游戏"。**这是自觉的升级,需用户正式点头一次**(升级不是背叛:把"AI友好"从教条变成"AI和人共同定义深度")。
- 未来时代的胜利条件具体设计(用户明确没想好)。

- 收益/社区/分发(当上行,不作地基)。

## 10. 诚实的总判断

- **宝石是真的**:"发育状态决定最优路线"的内生反滚雪球,是对文明六滚雪球病少有的、优雅的、且 AI 可验证的解法。
- **载重梁是真的**:人类玩家=深度评判标准+数据源,绕开了"先有强 AI"的死循环。
- **最大风险是 scope 和北极星对齐**:文明六规模对 AI-first 是自毁;必须靠"愿景做大、验证做小、P1.5 闸门、骨/皮砍法"控制。
