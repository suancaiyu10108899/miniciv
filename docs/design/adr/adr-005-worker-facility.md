# ADR-005：工人—设施经济模型

## 决策

工人每回合 1 操作（移动/建设施/在设施上生产）。资源产出来自**设施**而非直接来自地形。

## 否决的方案

**"站上去就产出"（旧项目式）**：工人站对应地形→自动+1资源/T。

否决理由：
1. 没有时间投入——工人到处跑就到处有产出，不需要策略
2. 工人死了→经济中断。设施持久→新工人可以接手旧设施
3. 建 vs 产的 tradeoff 天然创造了"投资窗口期"——建了一堆设施但还没收成时最脆弱

## 设施三规则

1. **建造**：工人在对应地形上花 1T 建造（平原→农场，森林→伐木场，山地→矿山）
2. **生产**：工人站在己方已完成设施上花 1T → 产出对应资源
3. **摧毁**：敌方近战单位站在设施格上→可选"摧毁设施"代替攻击

## 设计意图

- "铺设施"vs"守设施"vs"毁对方设施"——三种策略，需要 AI 学会取舍
- 设施持久→经济恢复力（死了工人还能重新采集，但需要新工人走到设施位置）
- 骑兵绕后毁矿山 → 不杀一个单位但对经济造成永久伤害
- 侦察兵发现对方矿山位置 → 有价值的情报
