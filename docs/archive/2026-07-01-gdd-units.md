# 2026-07-01 — GDD：单位 + 城市 + 经济 + 科技 + 迷雾

## 定下来的

### 单位
- 5 兵种：步兵、骑兵、弓箭手、侦察兵、工人
- 数值框架：HP + 攻击 + 防御 + 移速 + 视野 + 特殊能力
- 战斗：固定伤害 `max(1, ATK(+地形) - DEF(-地形))`，近战互打，远程白嫖
- 移动：骑兵 2 格平原/遇林停/禁山，步兵/弓/工人 1 格，侦察兵 2 格无视地形
- 克制三角：平原骑兵 > 步兵 > 山地步兵 > 骑兵，弓箭克制低防单位但被高防步兵盾反制
- 弓箭手远程不触发反击，不能占领城市
- 经验/升级不做

### 城市
- HP 500, DEF 25, ATK 0
- 占领规则：近战移动进城市格 → 伤害 ≥ 剩余 HP → 占领胜利
- 生产：四正方向邻格瞬间生产。天然限制一回合出兵量
- 基础产出：+1 粮/回合（即使工人全灭也能缓慢恢复）

### 经济
- 三资源：粮(平原)、木(森林)、金(山地)
- 工人站对应地形 → +1/回合
- 不同兵种不同资源组合：步兵(粮)，弓箭(粮+木)，骑兵(粮+金)
- 开局：粮 5 + 木 5 + 金 5，2 工人 + 2 侦察兵，**无初始战斗单位**
- 资源双方可见

### 科技
- 三分支：战斗/经济/建设（DAG，不互斥但研究槽位只有一个）
- 研究：花资源开启 → 城市执行(1-2T) → 完成
- 信息博弈：建设分支可见（= 项目进度），战斗/经济不可见但可推断

### 迷雾
- 三态：UNKNOWN / EXPLORED / VISIBLE
- v1 半径视野，v2 升级 LOS（和弓箭弹道共用同一套视线算法）

### 胜利
- 征服：占领城市
- 建设：5 个科技树项目完成

科技树的具体节点结构和数值填充。然后回头处理地形生成参数 + 先手平衡方案。
