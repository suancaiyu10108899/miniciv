# GDD — 城市

## 属性

| 属性 | 值 | 说明 |
|---|---|---|
| HP | 500 | |
| DEF | 25 | 骑兵(ATK 55)→30 伤/T。4 骑围攻≈4-5T 破城 |
| ATK | 0 | 城市不反击 |

## 占领规则

近战单位移动进城市格 → 计算伤害。若**此移动造成的伤害 ≥ 城市剩余 HP** → 占领成功 → 征服胜利。

- 弓箭手**不能占领**城市（可以削血，不能收人头）
- 城市格上的敌方近战单位 → 己方近战可以移进去攻击它
- 己方单位可以站在己方城市上防守（享受城市 DEF 加成 +25）

## 生产

单位在城市的**四个正方向邻格**（平原）瞬间生产。生产完成：
- 单位放在邻格空地
- 当回合即可移动
- 如果四邻全被占 → 不能生产（等下一回合）

天然限制：一回合最多出 4 个单位（且需要空地）。

## 城市基础产出

城市自身每回合 **+1 粮食**。工人产出是增量。
即使所有工人被杀，仍有缓慢恢复能力。

## 初始布局

城市周围 3×3：

```
山   平   林
平  [城]  平
林   平   山
```

- 四正方向 = 平原（单位自由出入）
- 四对角 = 山/林交替（初始三种资源各 2 格可采）

双方初始布局相同。

## 城墙（科技 C2）

完成科技 C2 → 城市 HP +100（总 600）。等效增加 ~3 轮围攻时间。
