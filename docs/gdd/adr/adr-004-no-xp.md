# ADR-004：不做经验/升级系统

## 决策

单位没有经验值，不能升级。

## 理由

1. **回合数太少。** 100 回合→单个单位可能参与 5-10 次战斗。没有足够的时间积累和体现成长曲线
2. **AI 复杂度。** 经验系统让每个同类型单位有不同的状态（老兵 vs 新兵）→状态空间翻倍。对 AI 编码和搜索都不友好
3. **雪球风险。** 先手方的单位先吃到战斗经验→升完级碾压后手方的同级单位→先手优势被经验系统放大

## 替代方案

科技树提供全局加成（+ATK/+DEF/+HP）而不是单单位升级。效果等同但所有同类型单位等强度——AI 状态编码不需要追踪每个单位的个性化成长。
