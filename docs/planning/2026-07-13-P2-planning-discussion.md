## 已达成共识的设计决策

### 游戏宏观框架
- **四时代**：远古（探图布局）→ 封建（领袖特色+聚落分化）→ 现代（红白分叉关键决策）→ 未来（冲刺胜利）
- **棋盘**：默认 25×25 六边环面，代码支持 20-40 范围
- **时代过渡**：中位玩家政策线到达节点 → 10 回合倒计时 → 共享同步换时代
- **封建→现代**：明牌展示红白选择
- **目标长度**：聪明玩家/AI 200-300T，新手/弱 AI 可拖到 500-600T，分数胜利兜底但正常不该触发

### 聚落系统
- **三级聚落**：村子(1格) → 城镇(3格/2建筑槽) → 城市(7格/4建筑槽) → 专业化城市(7格/6建筑槽)
- **专业化三选一不可逆**：工业重镇 / 金融中心 / 文化中心
- **粮食供应链**：城市净消耗粮食，需要村子/农业城镇供养
- **反爆铺靠效率非惩罚**：集中投资的农业城镇效率 > 铺大量村子

### 红白分叉与城乡联动
- **白线（城市-市场路径）**：城市产出×1.5，城市化加速，周期性市场危机
- **红线-苏联模式**：城市工人提供组织度，城市暴动
- **红线-中国模式**：农村根据地，民兵，土地改革事件，"农村包围城市"

### 贸易系统
- **依赖度机制**：每回合接受贸易 → 依赖度+1，断供后戒断反应（产出惩罚持续到依赖度归零）
- **自主替代**：花资源加速脱钩，完全自主后可自产
- **战略资源地理分布**：铁（北）/ 马（南）/ 稀土（东）——无人能自产一切
- **一带一路**：互利投资，对方可拒绝但 10T 内不能接受任何人投资，接受 = 囚徒困境

### 核武器
- **两弹一星**：核弹+导弹+卫星，需要稀土资源
- **三位一体**：陆基（慢但稳）/ 空基（快但脆）/ 海基（慢但隐蔽）
- **预瞄准**：已部署核弹预先指定目标，对手不可见
- **中断机制**：首枚核弹发射 → 所有已部署核弹玩家选择是否发射 → 同时命中
- **中国原则**：先手核打击天命-50+永久外交惩罚+支持度归零，反击惩罚轻得多

### 资源体系（五货币 + 三战略）
- **粮食**：养活人口、建新村、支撑专业城市 → 农场、村子、农业城镇
- **产能**：建造设施/道路/超级工程/核弹部署 → 矿山、伐木场、工业城市
- **财富**：维持军队、贸易投资、战后重建 → 市场、商路、金融中心
- **科技**：解锁硬科技节点（兵种/设施/产能） → 学院、图书馆、科技城
- **天命**：解锁软制度节点（路线/制度/外交） → 聚落、哲学流派、文化中心
- **铁**：骑兵/攻城器械/工业重镇 → 棋盘北部
- **马**：骑兵/驿道 → 随机分布，全图4-6个
- **稀土**：两弹一星/未来科技/高铁 → 棋盘东部

### 核心不做（砍掉的系统）
- ❌ 信仰/宗教 → 天命 + 诸子百家替代
- ❌ 奢侈品/宜居度 → 天命是唯一满意度指标
- ❌ 住房/人口 → 聚落等级隐式代表
- ❌ 城邦/使者/世界议会 → P3 再议
- ❌ 伟人/间谍 → FOW+侦察兵+卫星替代
- ❌ 政策卡（可更换） → 每时代一路线（不可逆固定选择）
- ❌ 独立文化树 → 一棵树两类节点（绿科技/蓝制度）
- ❌ 水坝/河流 → P3

### 开发体系
- **版本号**：V1.0.0（规划锁定）→ V1.x.0（Phase里程碑）→ V1.x.x（小里程碑/bug）→ V2.0.0（P2发布）
- **CLAUDE.md**：改造为新结构（接手友好版）+ 每次对话同步
- **文档分层**：活跃真相源（≤10文件） + 历史归档 + 参考
- **算力分级**：L0(单核开发) / L1(4-8核体检) / L2(全核-2扫描) / L3(全核全量)
- **子Agent规范**：任务模板 + 隔离机制 + 验收标准
- **UI**：Rust WASM + Web 前端（人类试玩+回放查看+数据浏览，同一前端）

## 第九、十轮共识（能源 + 多边贸易）

### 能源体系
- **能源作为工业重镇运转成本**：每工业重镇消耗 1 能源/回合
- **化石能源（煤/油）**：地理分布不均，煤天命-1/回合
- **清洁能源 = 结构性解放**：打破地理依赖的唯一路径
  - 太阳能：平原/沙漠格，+2能源/回合，天命+1，建造8-12T
  - 核能：城域内，+5能源/回合，需稀土，核泄漏风险
- **转型窗口期**：建造清洁能源期间仍依赖旧能源 → 最脆弱的博弈窗口
- **权力逆转**：转型完成 → 不再进口 → 出口国失去收入
- **红白能源分化**：白线化石倾向+转型难，红线清洁+转型快

### 多边贸易竞争
- 多玩家环境下"不出口=被其他出口国超越"——囚徒困境
- 不需要硬编码惩罚——靠多玩家存在自然产生竞争压力
- **EGTA 三层验证**：EGTA（无支配？）→ 开发者自玩 → 人类盲测
- **P2 贸易 AI 最低要求**：TradeBlind / TradeAverse / TradePragmatic / TradeStrategic / TradeAdaptive

## 第十一轮共识（河流 + 灾害 + 军民两用）

### 河流系统：水域格替代边标记
- **河流 = 水域格按线状排列**（不是格边界标记）
- 小河 1 格宽 → 可涉水渡河（消耗全部移动力）
- 中河 2 格宽 → 需 2T 渡河
- 大河 3 格宽 → 需船或桥
- **收益**：河流宽度成为战术变量、河面战斗可能、灌溉直观化、渡河选渡口成为决策
- **桥梁**：工人建造在水域格上，和修路逻辑一致
- **三峡大坝 P3**：合法则上基于河流水域格

### 灾害系统
- 灾害不是纯随机惩罚 → 是机会成本的考验（军队救灾 vs 守前线）
- **洪水**：水域格随机扩张 1 格（汛期），淹没相邻农田
- **干旱**：远离河流的区域农田产出 -80%
- **地震**：随机，山区概率更高，城市设施受损
- **救灾机制**：工人抢修（慢）/ 军队救灾（快但军事任务搁置）
- 积极救灾 → 天命+5 + 灾后重建 buff
- 不管 → 天命惩罚 + 人口流失

### 军民两用科技
- 同一科技在军事和民用两个场景都有用
- **动力浮桥**：军事=过河消除惩罚，民用=加速救灾抵达灾区
- **无人机**：军事=FOW下额外视野，民用=灾害预警（减少损失）
- **工程兵**：军事=建造/拆除桥梁路障，民用=抢修设施/加固堤坝
- **卫星（一星）**：军事=全图视野，民用=同一回合看到所有灾害位置
- 设计原则：研究这些科技 = 双倍价值，但不研究 = 两边都弱

## 第十二轮共识（地形价值重定义 + Flexible Zoning）

### 地形时间表：每种地形有自己的"power spike"时代
- **平原**：早期种田（粮食+产能）→ 中期稳定产出
- **森林**：木材+掩护 → 后期碳汇（如果做环境系统）
- **山地**：矿产+防守 → 后期稀有矿产+天眼选址
- **沙漠**：早期贸易通道 → 中期太阳能+石油 → 后期三北防护林改造绿洲
- **海洋**：早期渔业 → 中期海上风场+海上丝路 → 后期深海资源+航天回收
- **核心原则**：没有"差"地形。只有"需要不同开发路径"的地形

### 海洋开发
- 渔场（沿海格，+2粮食）、海上风场（沿海格，+2能源）
- 海上丝绸之路：港口 → 自动连接海上贸易网络（和陆上一带一路平行）
- 海上航天回收平台：超越胜利的关键一步
- 填海造陆：P3

### 中国天眼（FAST）
- 特殊设施，建造条件：被 ≥3 个山地格包围的平原/洼地格
- 效果：全局视野+3、天命+2/回合、科技研究速度+20%
- 地形约束 = 不是所有玩家都能建 → 创造"特殊地形争夺"

### Flexible Zoning（灵活划区）
- **城市不再是固定 7 格蜂窝 → 是玩家自选的连续格集合**
- 聚落规模：村子1格 / 城镇3-5格 / 城市5-9格（随时代增长）
- 约束：中心格距离 ≤ 2（19 格候选池），所有格必须 BFS 连通
- **形状本身就是策略**：沿河（灌溉+贸易）vs 环山（矿产+防御）vs 跨小河（桥梁必需）
- 棋盘：25×25 够 2 玩家，30×30 够 4 玩家
- 代码量 ~250 行（非重构——只是改城市创建和验证逻辑）

### 城市防守适配（不增加军队规模）
- **民兵系统**：城市区每格自动防御，ATK = 天命 × 0.3，不占单位数
- **区域控制（ZOC）**：己方单位控制相邻城市区格，敌方进入自动拦截
- **城墙设施**：关键格 DEF+15，ZOC 范围+1
- **攻城策略化**：攻击方选择目标格（打工业区 vs 农业区 vs 中心格）→ 不同后果
- 2-3 个战斗单位 + 民兵 + ZOC = 足够守住 7-9 格城市
