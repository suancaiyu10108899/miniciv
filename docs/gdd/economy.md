# GDD — 经济

## 设施系统

资源不来自"站在地形上"——来自**设施**。

| 设施 | 可建在 | 建造耗时 | 产出 | 说明 |
|---|---|---|---|---|
| 农场 | 平原 | 1T(工人建) | +1 粮/T | 基础粮食来源 |
| 伐木场 | 森林 | 1T(工人建) | +1 木/T | 弓箭手+科技+建设 |
| 矿山 | 山地 | 1T(工人建) | +1 金/T | 骑兵+科技 |

### 工人操作

工人每回合只能执行**一个操作**：

```
1. 移动 1 格
2. 建造设施（目标格必须是对应地形 + 无人占领 + 无己方设施）
3. 生产（工人必须站在己方已完成的设施上 → 本回合产出资源）
```

工人不能在同回合内移动+建造+生产。你必须选择"这回合建农场，下回合开始收成"还是"跑过去收已经建好的"。

### 设施持久性

- 设施一旦建造 → **永久存在**（除非被摧毁）
- 己方任何工人可站在设施上生产（不要求是建造者）
- 敌方单位站在设施格上 → 设施无法被己方工人使用（敌方并未摧毁它，只是干扰）
- **设施可被敌方近战单位摧毁**：近战单位站在设施格上 → 本回合可选择"摧毁设施"代替攻击 → 设施移除

### 设计意图

- "建新设施"vs"用现有设施生产"——工人时间分配的核心 tradeoff
- 设施是**可被攻击的经济资产**——骑兵绕后毁矿山比杀工人更伤
- 设施持久→即使工人被杀，新工人可以直接站在旧设施上恢复生产（比旧项目"工人=唯一经济来源"更稳健）

## 城市基础产出

城市自身每回合 **+1 粮**。独立于工人和设施。

### 建设胜利

科技 C5 纪念碑 = 30 粮 + 30 木 + 30 金。

## 生产机制

- 资源足够 → 在城市四邻格**瞬间生产** + 当回合可移动
- 四邻格必须有空位（己方/敌方单位占着→不能在那格生产）
- 无数量上限
- 维护费：v1 不做。原型跑 500 局后再评估

## 科技对经济的影响

| 科技 | 效果 |
|---|---|
| E1 耕作 | 农场产出 +1(→2 粮/T) |
| E2 伐木 | 伐木场产出 +1(→2 木/T) |
| E3 采矿 | 矿山产出 +1(→2 金/T) |
| E4 物流 | 工人移速 +1(→2 格/T)，可移动+建造(不可移动+建造+生产) |
| C4 粮仓 | 城市基础产出 +2 粮/T(→3 粮/T) |
