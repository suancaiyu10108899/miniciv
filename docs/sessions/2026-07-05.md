## 决定了什么

### 设计决策
- 方格终版参数：facility=4, HP=80, DEF=5, DMG=5, max_t=80
- 弓手不能占城暂不改变（先等新经济下弓手数据）
- 缺 DQN 兵种多样性（结构问题——生产不经过 NN，不是重训能解决）
- 缺 BC 完整 AI（等 Rust 后参数稳定）
- **坐标系决策未定**：方格数据已就绪，六边引擎可用等你评估

### 流程决策
- 子 Agent 适合机械翻译/格式化/纯读分析，不适合核心游戏逻辑
- 规划必须落定后执行（WORKFLOW 纪律条款）
- 矩阵被系统休眠杀了两次——后续可能要考虑进程守护
