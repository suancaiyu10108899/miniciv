## 1. 远古时代素材

### 1.1 生存与聚落

| 素材 | 历史背景 | 可能用法 |
|------|---------|---------|
| 仰韶/龙山/良渚文化 | 新石器时代三大遗址，分别代表黄河流域旱作（粟）、黄河流域晚新石器（黑陶）、长江流域稻作文明 | 远古开局可选"文明起点"：北方粟作（耐旱/稳定但产出低）vs 南方稻作（高产但需水利）vs 东方滨海（渔盐+航海早期优势）。不同起点 = 不同初始科技倾向 |
| 半坡遗址 | 环壕聚落，半地穴房屋，最早的防御工事 | 远古时代最基础的"城防"概念——环壕村落。被攻击时防御力低但建造成本极低 |
| 河姆渡干栏式建筑 | 长江流域湿热地带的高脚屋，防潮防虫蛇 | 地图上的湿地/沼泽格：普通聚落无法建造，需研发"干栏建筑"科技后才能在湿地上建村 |
| 井的发明 | 河姆渡已有人工水井，脱离地表水源限制 | 远古时代关键科技——"凿井"：解锁内陆无水格建村的能力，是聚落扩张的核心前置 |
| 刀耕火种 vs 耒耜 | 早期农业两种形态：火耕（烧林→肥田→弃耕游耕）vs 耒耜（木制翻土工具，定点耕作） | 远古核心经济选择：「游耕」产出高但每N回合需迁移（自动弃村），「定耕」产出低但稳定增长且可以积累改良 |
| 家畜驯化 | 猪（最早，约9000年前）、狗、鸡、牛、羊、马（传入较晚） | 远古畜牧线科技：猪→粮食转化率高，牛→耕作效率加成，马→移动速度加成（远古马极为珍贵） |
| 玉器文化 | 良渚玉琮、红山玉龙——早期社会的宗教/权力象征 | 远古"天命"前身——玉器产出。玉器格（稀有地形）产出"威望"资源，影响部落统一/联盟的形成 |

### 1.2 部落与战争

| 素材 | 历史背景 | 可能用法 |
|------|---------|---------|
| 炎黄与蚩尤 | 传说中三大部落联盟的战争：炎帝（农耕部落）vs 黄帝（游牧/半农部落）vs 蚩尤（九黎，铜兵器最早使用者） | 远古"三部落开局"剧本：玩家选择不同初始部落特性。蚩尤=最早铜兵器优势但外交孤立，黄帝=居中能联盟，炎帝=农业产出高但军事弱 |
| 禅让制 | 尧舜禹的传说：部落联盟领袖不传子，选贤 | 远古非世袭领袖机制：领袖更替时可在"贤能（高能力随机）"、"子嗣（低能力确定）"、"军功（军事强但可能内战）"中选 |
| 青铜兵器的出现 | 商周时期青铜戈/矛/钺/剑——从石器到金属的质变 | 远古→封建过渡的关键科技节点。解锁铜矿采集→青铜冶炼→青铜武器制造。注意：中国早期缺锡，青铜是奢侈品——不是所有兵都能换装 |
| 战车 | 商周车战——双马战车，三人（御者/弓手/戈手），贵族专享 | 远古晚期→封建早期精英兵种。战车需要"马"+"青铜"+"平原"三条件，是贵族单位的原型 |
| 甲骨文 | 商代占卜文字刻在龟甲兽骨上 | 远古"知识积累"机制：文字发明前知识靠口传（科技研究速度慢/有概率"失传"），文字发明后速度提升且不遗忘 |
| 方国体系 | 商代"内服外服"：核心区直辖，周边方国朝贡/联盟/敌对 | 远古→封建过渡期的"多势力博弈"：不是所有敌人都是敌人，方国可以是朝贡（定期纳贡换保护）、联盟（共同防御）、附庸（被动跟随宗主） |

|---------|---------|----------|

## 2. 封建时代素材

### 2.1 政治制度与治理

| 素材 | 历史背景 | 可能用法 |
|------|---------|---------|
| 封建制（分封） | 周天子分封诸侯：公侯伯子男五等爵，封地大小不同 | "封建"时代的命名机制：玩家在新占领土地上"分封"→创建半自治的附庸城。附庸提供贡赋但不完全听从命令。集权度（郡县 vs 分封）是核心决策轴 |
| 宗法制 | 嫡长子继承，大宗小宗，血缘网络 | 王朝稳定性机制：继承越合规（大宗嫡长），内乱概率越低；越违规（幼子/旁支篡位），爆发内战概率越高 |
| 礼乐制度 | 周礼——用礼仪等级代替暴力维持秩序 | 一种"非军事稳定手段"——投入资源搞礼乐→提升百姓支持度/减少叛乱概率，但军事产出下降 |
| 郡县制 | 秦废除分封，全国划郡县，中央派官，不可世袭 | 封建→帝国的制度选择：郡县（高控制力/高税收但官僚成本高）vs 分封（低成本/低控制力但附庸可能叛变）。这是你游戏红白二元的"远古版本" |
| 科举制度 | 隋唐创立，宋朝完善，通过考试选拔官僚 | 封建→帝国后期解锁的"精英选拔"制度：每N回合举行科举→产出高质量官员（提升城市效率）。副作用：形成士绅阶层（土地兼并加剧） |
| 九品中正制 | 魏晋选官制——按家世门第评级任官 | 科举的替代品/前身：门阀垄断官位→贫富分化加剧，但官员更忠诚 |
| 三省六部制 | 隋唐中央：中书出令→门下审核→尚书执行，六部（吏户礼兵刑工）分管 | 帝国时代的高级治理升级——解锁后城市管理效率大幅提升，但维护成本也高 |

### 2.2 农业与经济

| 素材 | 历史背景 | 可能用法 |
|------|---------|---------|
| 井田制 | 西周理想化土地制度：八家共耕公田（中间一块的产出归领主） | 封建早期默认经济：每城自动产出部分资源归中央（"公田"），其余归私。私人产出效率低但民怨少；提高公田比例→产出高但支持度降 |
| 初税亩 | 鲁国594BC按亩征税，承认土地私有 | 破除井田制的关键改革科技：大幅增加税收但引发贵族不满（叛乱风险临时升高） |
| 均田制 | 北魏→唐：国家按人口分配土地，死后归还 | "均田"政策——定期重新分配土地→贫富差距缩小，支持度提升，但生产效率惩罚 |
| 租庸调 | 唐代税制：租（粮）、庸（力役）、调（布帛）三合一 | 帝国成熟期的复合税制：玩家可在"实物/力役/货币"三种税收中自由选择配比 |
| 一条鞭法 | 张居正改革：各种赋税合并为单一银两税 | 帝国后期税制简化——管理成本大幅下降，但对银的需求暴增（如果银矿不足→财政危机） |
| 盐铁专营 | 汉武帝开始：盐和铁由国家垄断经营 | 国家垄断战略资源——盐（粮食加工必需）铁（武器必需）。玩家可决定哪些资源国营：国营=稳定收入但产出效率低，私营=高效率但税收少且可能被商人操控 |
| 交子（纸币） | 北宋四川民间→官交子，世界最早纸币 | 金融体系解锁——纸币可超发（短期获得大量资金但引发通胀→支持度崩盘），是一种高风险高收益的经济工具 |
| 钱庄/票号 | 明清金融：存款、放贷、异地汇兑 | 帝国晚期银行体系——解锁后可以"借未来钱"（透支资源），利息取决于"信用度" |
| 市舶司 | 唐宋元海外贸易管理机构 | 解锁海外贸易——建立港口城市，获得稀有资源（香料/象牙/玻璃）但受海盗/海禁影响 |

### 2.3 科技与工程

| 素材 | 历史背景 | 可能用法 |
|------|---------|---------|
| 铁器冶铸 | 春秋晚期→战国：铁制农具+兵器，取代青铜 | 战国军事经济分水岭：铁比青铜便宜且矿源广泛→军队规模大幅膨胀。从贵族小规模战争→全民动员的大规模战争 |
| 牛耕 | 战国：铁犁+牛耕→深耕效率翻倍 | 农耕效率质变科技，前置需要"铁器"+驯化牛 |
| 都江堰 | 李冰父子，公元前256年，无坝引水，至今运行 | 古代工程巅峰代表——可以对特定格进行"水利改造"（永久改变地形产出）。耗时极长但效果永久 |
| 郑国渠 | 韩国"疲秦"之计反而成就秦国——关中平原灌溉 | 水利工程 = 长期投资，花费巨大资源+多回合，完成后产出大幅提升。还有"为敌修渠反被利用"的戏剧性 |
| 灵渠 | 秦始皇连接湘江和漓江，沟通长江和珠江水系 | 运河系统——可在内陆格人工开挖水道，连通不同水系，降低运输成本 |
| 大运河 | 隋炀帝开凿京杭大运河，贯通南北 | 国家级超级工程——时代级项目，需要消耗海量资源但完成后永久改变经济地理：南北贸易效率翻倍 |
| 长城 | 从战国各国边墙到明长城，延续2000年 | 分段建造的边境防御体系——每段需N回合+N资源，建好后游牧民族无法从该段入侵。但维护成本高，且"筑城"意味着放弃主动出击 |
| 造纸术 | 蔡伦改进，公元105年 | 知识传播加速——科技研究速度翻倍，书籍可以"出口"给邻国赚取外交影响力 |
| 印刷术 | 雕版（唐）→活字（北宋毕昇） | 再次提升科研速度+解锁"意识形态传播"——可以往邻国输出文化影响 |
| 火药 | 唐末→宋军用（火箭/火炮/震天雷） | 从冷兵器到热兵器的过渡。注意中国火药早期主要用于烟火/信号/火攻，不是立刻改变战争形态——这是"渐进的质变" |
| 指南针 | 战国司南→北宋罗盘，用于航海 | 解锁远洋航行——海外贸易范围大幅扩大。也是FOW系统里的重要道具：有指南针的军队穿迷雾格不减移动力 |
| 浑天仪/地动仪 | 张衡，观测天体+检测地震 | 天文观测→历法精准→农业产出提升。地动仪→提前预警自然灾害减少损失 |

### 2.4 军事

| 素材 | 历史背景 | 可能用法 |
|------|---------|---------|
| 孙子兵法 | 春秋，孙武，"不战而屈人之兵" | 不是科技是"理念"——研发后解锁特殊战术选项：围城不攻（断粮迫使投降）、离间（策反对方将领）、避实击虚（有概率绕过敌军迎击） |
| 孙膑兵法 | 战国，围魏救赵、减灶增兵 | 军事欺骗战术科技——研发后解锁"佯攻"选项（假动作） |
| 武经七书 | 北宋汇编：孙子/吴子/司马法/六韬/尉缭子/三略/李卫公问对 | 军事学术集大成——可以当"军事理论累计"的科技链 |
| 魏武卒 | 吴起训练的精锐重步兵，选拔极严 | 精锐兵种原型：不是谁都能造，需要特定条件是"人口≥X + 支持度≥Y"。生产成本极高但战斗力远超普通步兵 |
| 秦弩 | 标准化弩机（望山/悬刀/牙），零件可互换 | 远程步兵升级——弩 vs 弓：弩杀伤高但射速慢（每2回合攻击一次），弓射速快但杀伤低。秦弩的"标准化"意味着生产成本降低 |
| 诸葛连弩 | 半自动连射弩，一次可发十箭 | 精锐远程兵——极贵，但一回合可攻击多个目标（AOE） |
| 骑兵革命 | 赵武灵王胡服骑射（公元前307）→后来骑兵成为决定兵种 | 从战车到骑兵的关键转变——需要"胡服骑射"改革（政治成本：贵族反对），完成后骑兵淘汰战车 |
| 马镫 | 大约4-5世纪传入中国，彻底改变骑兵战斗方式 | 重骑兵科技——马镫之前骑兵只能骑射/骚扰；马镫之后可以马上冲锋。对骑兵战斗力质变 |
| 火器（早期） | 突火枪（南宋）→火铳（元明）→红衣大炮（明末） | 火器是渐变不是突变——早期火器精准度低、装填慢、怕雨，要和冷兵器混编。直到红衣大炮才能真正攻城（城墙不再无敌） |
| 戚继光/鸳鸯阵 | 明代对抗倭寇的12人小组战术混合冷热兵器 | "阵法"系统——不同兵种混编有协同加成。鸳鸯阵 = 盾牌+长枪+狼筅+刀牌+火枪的战术配置 |

### 2.5 文化思想

| 素材 | 历史背景 | 可能用法 |
|------|---------|---------|
| 儒家（孔子） | 仁/礼/忠恕/中庸 | 封建/帝国时代主要治理哲学之一：提高支持度/降低叛乱率，但军事效率-10%、科技偏保守。"文化胜利"路径之一 |
| 法家（商鞅/韩非） | 法治/耕战/赏罚分明 | 高效率军事机器：军功授爵（杀敌换晋身）、重农抑商、法治严苛。产出高+军队强但支持度偏低且容易民变 |
| 道家（老子/庄子） | 无为而治/顺应自然 | 低干预治理：税收低、军备弱，但人民自发创新（随机科技加速）且自然灾害恢复快 |
| 墨家（墨子） | 兼爱/非攻/尚贤/科技 | 反战+科技路线：解锁高级防御工事（墨家守城术）、早期光学/力学。墨家机关——攻城器械/守城器械专精 |
| 兵家（孙武/吴起等） | 军事哲学（不是兵种技术） | 上面已涵盖 |
| 纵横家（苏秦/张仪） | 合纵连横，外交谋略 | 解锁高级外交选项：合纵（联合多国对抗强国）、连横（强国分化瓦解联盟）、远交近攻 |
| 农家（许行） | 君民并耕/重农 | 极端农业路线：领袖与民同耕→支持度极高但官僚系统退化 |
| 名家（公孙龙） | 白马非马/逻辑辩论 | 偏学术趣味——解锁"辩论"文化事件，可能小概率获得意想不到的科技突破 |
| 阴阳家（邹衍） | 五德终始/五行 | 历史命运理论——影响领袖更替时的合法性论述 |
| 诸子百家选择 | 玩家在封建时代必须从诸子百家中选一家作为"国教" | **互斥的永久选择**——类似你红白分叉的远古版本：法家=军事强权/儒家=稳定发展/道家=弹性顺应/墨家=防守科技。选了就不能改，影响后续整个游戏走向 |
| 佛教传入 | 汉代传入→魏晋南北朝鼎盛 | 外来思想——接受佛教→解锁寺庙建筑（支持度+文化产出），但同时引发本土儒家/道家反弹（短期不稳定） |
| 宋明理学 | 朱熹/王阳明——儒家哲学化/心性化 | 儒家的升级版——解锁后支持度上限提高，但社会趋于保守 |
| 焚书坑儒 vs 独尊儒术 | 秦用法家禁百家 → 汉武帝独尊儒术 | 意识形态管控——可以强行"统一思想"→稳定度大幅提升但科技速度下降（学术自由被压制） |

## 3. 帝国时代素材（唐宋元明清）

### 3.1 疆域与治理

| 素材 | 历史背景 | 可能用法 |
|------|---------|---------|
| 羁縻制度 | 唐代对边疆民族"因俗而治"：保留当地首领地位，名义归属中央 | 帝国时代边疆管理模式：对征服地区可选"直辖"（同化成本高但长期收益）或"羁縻"（低成本低收益但可能反叛） |
| 土司制度 | 元明清西南少数民族地区世袭土官 | 羁縻的进阶版——更系统化的间接统治 |
| 理藩院 | 清代专管蒙古/西藏/新疆事务的中央机构 | 帝国治理大疆域的官僚科技——解锁后边疆管理效率提升 |
| 改土归流 | 清代废除土司，设流官 | 从间接统治→直接统治的最后一步——高风险高回报 |
| 朝贡体系 | 东亚国际秩序：周边国家向中国朝贡获得贸易权和保护 | 帝国时代核心外交机制：不是征服，而是建立朝贡关系——朝贡国定期纳贡，宗主国提供军事保护。朝贡国多→天命提升 |
| 藩属国 | 朝鲜/越南/琉球——不完全独立也不完全吞并 | 介于"独立国"和"行省"之间的中间态——投入少但控制弱 |
| 和亲 | 汉唐用公主联姻换取边境和平 | 外交婚姻——牺牲一个贵族女子+一些嫁妆→换取与邻国N回合和平 |

### 3.2 贸易与经济全球化

| 素材 | 历史背景 | 可能用法 |
|------|---------|---------|
| 陆上丝绸之路 | 汉→唐：长安经河西走廊/西域→中亚→波斯→地中海 | 商路系统——地图上生成"丝绸之路"路径格，在路径上建驿站→获得贸易收入。但路径上的安全需要军队保护 |
| 海上丝绸之路 | 唐宋→明：泉州/广州→东南亚→印度→阿拉伯→东非 | 海洋商路——类似但走海路。贸易收入更高但需要造船技术+海军护航 |
| 郑和下西洋 | 1405-1433，七次远航至东非 | 帝国时代的"国家远航"——不是常态化贸易，而是皇家舰队出航（消耗巨大→政治圈反对→明朝后来停了）。玩家面临：继续远航（消耗资源换外交影响力+稀有物资）vs 停止巡海（省钱但失去影响力） |
| 朝贡贸易 | 朝贡使团带贡品来+皇帝赏赐更多回去 | 亏本买卖——换的是政治面子（天命/威望）不是经济利益 |
| 广州十三行 | 清代唯一的对外通商窗口 | 海禁下的特殊贸易——只开一个港口，收入有限但可控 |
| 白银流入 | 明中叶→西属美洲白银经菲律宾→流入中国 | 被动的全球化影响——你不是主动参与者但受世界贸易影响：白银多了（通胀/购买力变化），白银少了（通缩/经济危机）|
| 鸦片贸易 | 19世纪英国用鸦片逆转贸易逆差→白银外流→鸦片战争 | 警惕素材——可以作为"敌国用非军事手段削弱你"的设计原型 |

### 3.3 城市化与社会

| 士农工商 | 传统四民等级 | 社会阶层系统：不同阶层有不同的税收贡献/支持度/军事动员率。玩家可通过政策调整阶层结构 |
| 流民/起义 | 历代王朝末年：黄巾、黄巢、李自成、太平天国 | 帝国内部崩溃机制——支持度太低→流民→起义军→可能推翻王朝（不是你输给别国而是输给人民） |

## 6. 跨时代系统素材

### 6.1 可跨时代应用的"中国概念"

| 势 | 趋势/大势——顺势而为事半功倍 | "时代潮流"概念——某些行为顺应时代加成效，某些逆时代有惩罚 |

| 瓷土 | 景德镇高岭土 | 高端瓷器产地——特殊瓷土格产出"瓷器"奢侈品 |
| 茶山 | 江南/福建/云南 | 茶=后来居上的大宗出口品（18-19世纪全球贸易核心） |
| 石油 | 大庆/新疆/渤海 | 工业时代必需 |
| 稀土 | 内蒙古/江西 | 现代/未来高科技必需——你已讨论过 |
| 水力 | 河流经过的格 | 水车/水磨→早期工业动力；水电站→现代能源 |
| 风力 | 沿海/高原 | 可再生能源——未来时代的清洁能源之一 |
| 太阳能 | 西北荒漠 | 同上，不同地形有不同能源优势 |
