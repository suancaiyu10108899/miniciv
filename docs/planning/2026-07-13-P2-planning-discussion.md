## 第一轮讨论回顾（已完成）

### P2 游戏设计目标（草案）

- 棋盘：25×25 六边环面（625 格）
- 城市：2 城/玩家
- 兵种：5-7 种
- 科技：18-22 节点 DAG
- FOW：单位视野 + 地形影响
- 红白分叉：T30-50 可选
- 多回合建造/生产
- 目标：100-200T 自然游戏长度，<15% 阶梯率

### P2 分阶段（草案）

```
Phase 0: 地基清理
Phase 1: 棋盘扩张 (25×25 + 多城)
Phase 2: 深度机制 (FOW + 科技树 + 多回合)
Phase 3: P2 甜点搜索
Phase 4: AI 适配
Phase 5: 人类可玩 (UI + 试玩)
Phase 6: 收尾
```
