# ADR-003：固定伤害（非随机）

## 决策

战斗伤害 = `max(1, ATK+地形 - DEF-地形)`，无随机波动。

## 理由

1. **AI 可精确计算。** FlatMC rollout 中每一步战斗结果确定→不需要增加模拟次数来平均随机波动
2. **干净信用分配。** RL 训练中每一步的后果是确定的→信用分配信号干净
3. **策略可预测。** 玩家/AI 知道"3 刀杀"vs"4 刀杀"是确定的→可以用血量做精确计算（"我剩 61 HP 能吃 2 刀"）

## 否决随机

随机伤害让 FlatMC rollout 需要多倍模拟才能收敛到期望值。在 50 回合 horizon 下，累积随机波动会淹没策略差异。

## 保底机制

`max(1, ...)` — 即使防御高于攻击力，也至少扣 1 HP。避免"完全打不动"导致死局。
