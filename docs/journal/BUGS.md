# Bug Journal

> 每个 Bug 一条记录。格式：日期 + 现象 + 根因 + 复现条件 + 修复方案 + 预防措施。
> 不迁移旧项目的 Bug——那些是旧代码的。从新的开始记录。

---

## 模板

```markdown
### B00X: 简短标题

**日期**: YYYY-MM-DD
**严重度**: CRITICAL / HIGH / MEDIUM / LOW
**发现者**: 人 / 主Agent / 子Agent / 实验数据

**现象**: 发生了什么？（具体、可观测）

**根因**: 为什么发生？（代码层面、设计层面）

**复现条件**: 怎么触发这个 Bug？

**修复方案**: 怎么修的？

**预防措施**: 以后怎么防止类似问题？

**关联**: 相关 commit / issue / insight
```
