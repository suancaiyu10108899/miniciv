## 战斗规则

### 伤害公式

```
damage = max(1, attacker.ATK + terrain_ATK_bonus - defender.DEF - terrain_DEF_bonus)
```

- **固定值，不随机**
- 近战双方互打 → 各自扣血
- 远程（弓箭手）：攻方输出，守方不还手
- 骑兵冲锋：走 2 格平原后攻击 → +10 ATK

### 击杀回合数目标

| 对战 | 目标 | 验证 |
|---|---|---|
| 骑兵冲锋 vs 平原步兵 | 2-3T | 骑 ATK 65 - 步 DEF 30 = 35 → 100/35≈3T ✓ |
| 骑兵 vs 山地步兵 | 骑兵打不了(禁山) | 设计意图：地形=绝对壁垒 |
| 步兵 vs 步兵(平原) | 4-5T | ATK 40 - DEF 30 = 10 → 100/10=10T ✗ 需要调 |
| 弓箭 vs 平原骑兵 | 2-3T | ATK 45 - DEF 15 = 30 → 80/30≈3T ✓ |
| 弓箭 vs 山地步兵(防45) | 6-8T | ATK 45 - 45 = 5 → 100/5=20T ✗ 需要调 |

> 带 ✗ 的行说明数值需要原型时校准。目标不是精确数字——是"这个组合能不能产生有意义的策略选择"。

### 回血

站在己方城市上 → 每回合 +n HP（具体值原型时调，初步 +5/T）

### 不做

- 经验/升级（回合太少，没空间堆叠经验值）
- 士气/状态效果
- 暴击/随机

## 移动规则

| 兵种 | 下次数 | 地形限制 |
|---|---|---|
| 步兵 | 1 | 无（可上山） |
| 弓箭手 | 1 | 无（可上山） |
| 工人 | 1 | 无（可上山） |
| 骑兵 | 2 | **禁入山**。第 1 格是森林→停(本回合结束) |
| 侦察兵 | 2 | **无视所有地形限制（可上山）** |

环面移动：坐标 wrap。`new_x = (x + dx) % W`。

## 克制三角

```
平原骑兵 ──碾压──→ 平原步兵
    ∧                │
    │                ↓ (上山后反向碾压)
    │            山地步兵
    │                │
  (3箭杀)           │ (防高→弓箭打不动)
    │                ↓
    └── 弓箭手 ←──────┘
```

这不是硬编码——是 ATK/DEF/移动限制/射程的自然数学交互。
