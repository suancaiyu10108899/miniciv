# AI 范式验证报告

## 测试的范式

| 范式 | AI | 训练方式 | 训练量 |
|------|-----|----------|--------|
| 手写启发式 | Greedy v1-v4 | 人工设计 | 0 |
| 进化算法 | Evo50 | 50代 × 60个体 | 45,000局 |
| 行为克隆 | BC | 监督学习(Greedy数据) | 870k样本 |
| 深度Q网络 | DQN | 自对弈+ε-greedy | 2,000局 |
| 自对弈训练 | SelfPlay Gen1-3 | 迭代自对弈 | 3,000局 |
| 规则+参数混合 | Hybrid | Evo调参50代 | 15,000局 |
| 搜索(FlatMC) | FlatMC v3-14 | 无训练 | 0 |

## 逐范式分析

### 手写启发式 ✅ 有效但上限低
- v1(简单贪心)是最好的人工设计版本
- 每次增加复杂度都降低性能
- 上限: vs Random 99%，镜像 54%

### 进化算法 ✅✅ 最成功
- Evo 4代收敛，89% vs Greedy
- 训练成本极低（<5分钟）
- 可解释（权重值有直观意义）
- **推荐作为默认 AI 训练范式**

### 行为克隆 🟡 有效但不完整
- 学会预测 Greedy 动作（49%准确率，21类）
- 策略极端化：只研究C5，放弃军事
- vs Greedy 100%胜（因为Greedy给了BC时间）
- vs Random 0%胜（Random的随机进攻让BC无法应对）
- **关键教训**：BC学到了"老师"的弱点而非优点

### DQN 🟡 有效但需验证
- 训练胜率92%，但成绩异常高
- 2,000局训练不足以学到复杂策略
- 可能和BC一样走了C5 rush捷径
- **需要独立验证**：DQN vs Evo50 直接对比

### 自对弈 ❌ 无改善
- Gen1/Gen2/Gen3 均96.5%（训练中）
- 迭代训练未产生进步
- 可能原因：每代用前代做对手 → 策略未多样化 → 陷入局部最优

### 规则+参数混合 ✅ 可行
- 规则提供骨架，参数提供灵活性
- 80%训练胜率
- 可解释性最好
- 适合作为"可调试AI"

### FlatMC ❌ 不适用
- 蒙特卡洛搜索对15×15无价值
- 单步移动的影响在搜索噪声中消失
- 可能适用于更大棋盘（30×30）的战略级决策

## 平台验证结论

**miniciv 是一个有效的 AI 训练平台**，特别适合：
1. **进化算法** — 最成功的范式（几分钟训练 → 超越手写AI）
2. **参数化决策** — 15个权重即能编码有效策略
3. **监督学习** — BC能学会行为模式（但需要更好的数据）
4. **强化学习** — DQN有初步效果（需更多验证）

不适合：
1. **蒙特卡洛搜索** — 动作粒度太细
2. **自对弈（当前形式）** — 缺少策略多样性机制
