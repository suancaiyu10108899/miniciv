## 机制分析

真实的 P0 效应来自三个维度：

1. **行动先手**：P0 单位先动 → 先占领有利地形 → 先攻击
   - 交替先手已解决（奇数 P0 先/偶数 P1 先）
   
2. **经济先手**：P0 先生产 → 先获得资源 → 先造出单位
   - 研究 tick 同时化已部分解决
   
3. **位置先手**：P0 单位初始位置可能更有利
   - 对称地图（balanced）下双方条件相等
   - 非对称地图下：P0 优势或劣势随机分布

**当前结论**：真实的 P0 效应在 tiebreak 修复后接近 0（48-52% 范围内），无需进一步修复。
