## 1. P2 的定位与边界

### 1.1 P2 是什么

P2 是从"最小原型验证红白线深度"到"一个完整、可玩、值得对外展示的六边形策略游戏"的跨越。

P1.5 证明了：
- 红白分叉的张力存在 ✅
- FlatMC 深度曲线无平台期（搜索空间饱满）✅
- 非传递克制环存在 ✅
- 但 15×15 单城的空间已被穷尽 ❌
- 所有 AI 都假设全信息（侦察兵是死兵种）❌
- C1 甜点参数是用乘数 hack 模拟的游戏长度，不是结构性 ❌

**P2 的任务：用结构性设计替代乘数 hack，产生一个真正能让人类觉得"每一步都在做有意义的决策"的游戏。**

### 1.2 P2 不是什么

- 不是完整的文明六替代品（那是 P3+）
- 不是要一次性实现素材库里所有机制
- 不是追求"中国历史模拟"——是追求"以中国历史为灵感的好策略游戏"
- 不是所有 AI 都必须适配所有机制——允许部分 AI 退化/简化

### 1.3 核心约束

| 约束 | 内容 |
|------|------|
| 棋盘 | 默认 25×25，代码支持 15-40 |
| 玩家数 | 2-8（1v1 / 2v2 / 4v4 / FFA 可配置） |
| 时代数 | 4（远古→封建→现代→未来）|
| 游戏长度 | 目标 150-300T（聪明玩家），最大 500T（分数胜利兜底） |
| 技术栈 | Rust 引擎 + WASM 前端 + 可选 Python 训练辅助 |
| 确定性 | 延续 P1.5 的确定性铁律（给定 seed → 完全相同结果） |
| 测试 | 每个新系统 ≥2 个单元测试 + 1 个集成测试 |

## 2. 宏观框架：四时代

### 2.1 时代概览

```
远古 ──→ 封建 ──→ 现代 ──→ 未来
(探索)  (发育+选择) (分裂)  (兑现)
 ~40T    ~80T      ~100T   ~80T
```

| 时代 | 主题 | 核心问题 | 关键机制解锁 |
|------|------|---------|------------|
| **远古** | 探索与奠基 | "我在哪？周围有什么？我怎么活下来？" | 地图探索(FOW)、基础农耕、聚落雏形、文字发明、邻近部落接触 |
| **封建** | 发育与选择 | "我是什么样的文明？" | 诸子百家（5选1不可逆）、聚落升级、铜铁冶炼、基础战争、长城/运河萌芽 |
| **现代** | 分裂与抉择 | "领先还是落后？走哪条路？" | 红白分叉（处境决定路线）、工业化、火器普及、贸易体系、全球外交 |
| **未来** | 兑现与超越 | "我赢了——但赢的方式是什么？" | 太空/核聚变/AI/基因/信息战，各线独特的胜利路径 |

### 2.2 时代过渡机制

**触发条件**：当所有存活玩家中位数完成该时代"关键科技/制度节点"的一定比例（例如 70%），时代过渡开始。

**倒计时**：过渡期持续 8-15 回合（取决于游戏速度设置），期间：
- 旧时代科技研究成本递减（知识扩散）
- 新时代科技开始变得可研究
- 所有玩家收到"新时代即将到来"的警告

**同步进入**：倒计时结束后所有玩家同时进入新时代。先进入者不能提前获得新时代科技（防止滚雪球依赖时代抢先）。

**时代加成**：每个新时代，所有玩家获得一次性的"时代红利"——少量资源注入。但落后者获得的红利略多于领先者（温和的追赶机制，不是橡皮筋）。

### 2.3 时代特有机制

| 时代 | 独有机制 |
|------|---------|
| 远古 | 部落探索（侦察兵可在中立格触发现"原始村庄"→一次性奖励）、猛兽事件（非玩家中立单位在地图上出现）、刀耕火种 vs 定耕选择 |
| 封建 | 诸子百家选择、贵族制度（世袭官僚→决策受门阀牵制）、朝贡贸易萌芽、铜→铁过渡 |
| 现代 | 红白分叉、意识形态传播（向邻国输出/被输入）、工业化污染（产能格/工业城产生污染→支持度惩罚）、殖民/反殖民动态 |
| 未来 | 太空竞赛、核威慑（如果还没用它赢）、AI 自治度（AI 接管部分城市管理→效率高但可能失控）、星际殖民（新地图类型！） |

## 3. 棋盘与地图系统

### 3.1 基础参数

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| 棋盘尺寸 | 25×25 = 625 格 | 15×15 到 40×40 | 六边形轴向坐标，环面拓扑 |
| 城市数量 | 2-3/玩家 | 可配置 | 初始城市在地图上分布（尽量对角/分散） |
| 资源分布 | 地理偏向 | — | 铁在北方，马在草原，稀土在东部等 |
| 地形比例 | 可配置 | — | 不同"地图剧本"有不同的地形分布 |

### 3.2 地形类型扩展

在现有 `Plain / Forest / Mountain / Water / City` 基础上扩展：

| 新地形 | 特性 | 产出基础 | 特殊 |
|--------|------|---------|------|
| **沙漠** | 移动力消耗+1，不能建普通农场 | 0/0/0 | 石油（现代发现）、太阳能（未来） |
| **草原** | 骑兵+1移动 | 1粮0木0金 | 马匹资源格概率↑ |
| **丘陵** | 防御+5 | 0粮1木0金 | 适合采矿 |
| **湿地/沼泽** | 移动力消耗+2，疾病风险 | 0粮0木0金 | 需排水科技后才能开发，石油概率↑ |
| **河流** | 经过的格+粮 | 每经过一格的农场+1粮 | 天然商路/运输线 |
| **海岸** | 海陆交界 | 0/0/1金（渔） | 港口必需 |
| **深海** | 需船只才能通过 | 0/0/0 | 航海科技解锁 |
| **火山** | 周期性喷发（灾难） | 喷发后极度肥沃 | 高风险高回报 |
| **绿洲** | 沙漠中的生命之源 | 3粮0木0金 | 极为珍贵 |
| **冻土** | 产出低 | 0粮0木0金 | 未来可能有资源 |

### 3.3 资源分布逻辑

战略资源不是均匀随机分布，而是有地理偏向——这创造"地缘政治"：

| 资源 | 偏向分布 | 稀缺度 | 时代 |
|------|---------|--------|------|
| 铜 | 中原/山地 | 中 | 远古→封建 |
| 锡 | 南部山地（云南型） | 极高 | 远古→封建（青铜的命门） |
| 铁 | 广泛但北方更多 | 中 | 封建→现代 |
| 马 | 北方草原/西部高原 | 中高（农耕区稀缺） | 全时代 |
| 盐 | 内陆盐池/沿海 | 中 | 全时代 |
| 瓷土 | 特定格（如景德镇型） | 高 | 封建→现代 |
| 丝绸 | 黄河/长江流域 + 桑树格 | 中高 | 封建→现代 |
| 茶叶 | 南方山区 | 中 | 封建→现代 |
| 煤 | 北方（山西型） | 中 | 现代 |
| 石油 | 沙漠/海洋/草原 | 中高 | 现代→未来 |
| 铀 | 随机分布 | 极高 | 现代→未来 |
| 稀土 | 东部（内蒙古/江西型） | 极高 | 未来 |
| 水力 | 河流经过的丘陵/山区 | 低（但可再生） | 全时代 |

**设计意图**：没有任何一个出生地能自给所有战略资源。你总要贸易、征服或替代方案。

### 3.4 地图剧本

提供多种预设地图生成器：

| 剧本 | 特点 |
|------|------|
| 平衡 | 每个玩家周围资源基本对称 |
| 大陆 | 一到两块大陆（海洋分割），航海重要 |
| 盘古 | 一块超级大陆，陆地战争为主 |
| 群岛 | 分散岛屿，航海/海军优先 |
| 河谷 | 沿河流分布的可居住区，其余为沙漠/山地 |
| 真实中国 | 模拟中国地理：西部高原→中部山地→东部平原，黄河长江两条生命线 |

### 3.5 地图改造

某些超级工程/科技可以永久改变地图格的地形：

| 改造 | 条件 | 效果 |
|------|------|------|
| 植树造林 | 科技 | 沙漠→平原（多回合） |
| 排水 | 科技 | 湿地→平原 |
| 水利 | 科技+河流 | 永久提升邻格粮产 |
| 开凿运河 | 超级工程 | 人工水道 |
| 填海 | 未来科技 | 浅海→平原 |
| 沙漠改造 | 未来超级工程 | 沙漠→可耕格 |

## 4. 聚落系统

### 4.1 聚落层级

这是取代 P1.5 单城系统的核心结构升级：

| 层级 | 规模 | 建筑槽 | 人口 | 粮食消耗/回合 | 升级条件 |
|------|------|--------|------|-------------|---------|
| **村子** | 1 格 | 0 | 1-3 | 自给自足 | 初始（工人可建） |
| **城镇** | 中心 1 格 + 周围可用格 | 2 | 4-10 | 净消耗 1 粮 | 人口≥4 + 花资源升级 |
| **城市** | 中心 1 格 + 周围可用格 | 4 | 10-25 | 净消耗 3 粮 | 人口≥10 + 花资源升级 |
| **专业化城市** | 同上 | 6（多2槽） | 不变 | 不变 | 城市 + 选专精方向 |

**关键**：城市（及以上）是**粮食净消耗者**——必须由周围村子/农业城镇供养。这创造了"城乡供应链"的真决策。

### 4.2 专业化方向（城市可选，不可逆）

| 方向 | 效果 | 最好的选择时机 |
|------|------|-------------|
| **农业重镇** | 粮食产出×2，可供应 2 个城市 | 早期扩张期 |
| **矿业重镇** | 木/金产出×1.5，可发现周边隐藏矿脉 | 工业化前夕 |
| **工业重镇** | 产能产出×2，建造速度翻倍 | 军事扩张期 |
| **商业中心** | 金产出×2，商路收入+50% | 贸易/和平路线 |
| **文化中心** | 天命产出×2，意识形态传播范围+2 | 文化胜利路线 |
| **科技城** | 科研速度×2 | 科技胜利路线 |
| **军事要塞** | 防御+50%，驻军维护费-50% | 前线边境 |

**设计意图**：专业化 = 路径锁定。你想建的每一种都想要，但必须选一个。这个决策的深度在于"我到底该怎么赢？"

### 4.3 城乡供应链

```
村子（粮+3/回合）
  → 自己消耗 2 → 余粮 1 → 输送至城镇
城镇（粮-1/回合）
  → 从周边村子接收余粮
城市（粮-3/回合）
  → 必须从 N 个村子/农业城镇获得 ≥3 净粮
  → 否则：人口下降，产出减半
```

**供应链距离**：村子→城镇→城市的距离越远，运输损耗越大（每 5 格距离损耗 10% 粮）。这创造"城市规划"的空间博弈——你的农业带需要紧邻城市带。

**红白城乡联动（扩展现有 P1.5 设计）**：
- **白线城市**：产出×1.5（已有），但消耗粮食也×1.5（城市膨胀→周边农业带压力大）
- **红线城市**：城市工人提供组织度（已有），但城市产出不享受白线加成
- **红线乡村**：农村根据地（新！）：红线可以在村子格组织"民兵"——村民在敌人入侵时自动转化为战斗力（人民战争）。村子支持度产出×2

### 4.4 人口与增长

人口自然增长率取决于：
- 粮食盈余（越多越快）
- 支持度（越高越快）
- 医疗/卫生科技（解锁后加速）
- 战争损耗（每场战斗死亡部分人口）

**人口迁移**：可以花费资源将一个聚落的部分人口迁移到另一个（用于加速新城发展或疏散边境居民）。

## 5. 资源与货币体系

### 5.1 五货币

| 货币 | 产出源 | 主要消耗 | 特殊 |
|------|--------|---------|------|
| **粮食** | 农场/渔场/村子/农业城镇 | 养人口、建新村、军队口粮 | 有"腐损率"——存粮每回合按比例损耗（除非有粮仓科技） |
| **产能** | 矿山/伐木场/工业城市 | 建造/修路/超级工程/军队装备 | 建造速度的核心约束 |
| **财富（金币）** | 市场/商路/金融中心/税收 | 军队维护、贸易投资、外交贿赂 | 也是"通胀"的载体 |
| **科技（灯泡）** | 学院/图书馆/科技城 | 研究硬科技节点 | 可"出口"——卖给落后国家换金币 |
| **天命（软实力）** | 聚落/诸子百家/文化中心/超级工程 | 解锁制度节点、意识形态传播 | 同时也是"国家稳定性"的衡量 |

### 5.2 三战略资源（不消耗，但拥有才能造对应单位/建筑）

| 战略资源 | 解锁内容 | 分布 |
|---------|---------|------|
| **铁** | 重步兵/骑兵重装/攻城器械/工业建筑 | 北方山地 |
| **马** | 骑兵/驿道/商队速度 | 草原/高原 |
| **稀土** | 未来科技/两弹一星/卫星/电子战 | 东部 |

**战略资源的"拥有"定义**：你的领土内至少有一个已开发的该资源格 + 一条连接到首都的路径（贸易路线或国内商路）。

### 5.3 货币间的兑换

不是随意自由兑换——需要解锁特定的制度/科技：

| 兑换方向 | 解锁条件 | 汇率 |
|---------|---------|------|
| 财富→科技 | 资助学术（封建科技） | 投入金获得科技加速 |
| 粮食→财富 | 市场经济（现代制度） | 出售余粮 |
| 产能→天命 | 公共工程（封建制度） | 建文化建筑 |
| 天命→科技 | 西南联大（红线专属） | 组织度兑换科技 |
| 天命→产能 | 集中力量（红线专属） | 组织度兑换产能爆发 |

**设计意图**：不同路线有不同的"兑换效率"。白线擅长财富→一切（市场万能），红线擅长天命→爆发（组织度兑换）。

### 5.4 通货膨胀

引入"物价"概念——金币太多 + 产能跟不上 → 单位/建筑成本上涨。这是对"无限印钞"的约束，也是金融管理成为真决策的原因。

## 6. 科技与制度双树

### 6.1 双树设计

```
硬科技树（Tech Tree）          软制度树（Institution Tree）
用"科技"研究                    用"天命"解锁
解锁：兵种/设施/产能提升        解锁：治理方式/外交手段/路线选择
唯物主义/可量化                 "软"的/意识形态/不可逆
```

**为什么是双树？**
- 单树的问题是"科技 → 一切"。现实中，技术能力和制度能力是不同的维度——你有火药不代表你能高效组织火枪队（明朝问题）。
- 双树创造交互效应：科技快但制度落后 → 有钱有枪但国家不稳定。制度先进但科技落后 → 清明但挨打（晚清的症结）。

### 6.2 硬科技树（每个时代 6-8 个节点）

#### 远古科技（6 节点）

| ID | 名称 | 解锁 |
|----|------|------|
| A1 | 用火 | 烧林（临时肥沃格）、驱兽 |
| A2 | 耒耜/犁 | 农耕产出+50%，与"游耕"互斥 |
| A3 | 凿井 | 可离河建村 |
| A4 | 陶器 | 粮储损耗-50% |
| A5 | 驯化（猪/牛/马） | 畜牧产出 / 牛耕加速 / 侦察兵移动+1 |
| A6 | 青铜冶炼 | 青铜武器（ATK+5），需要铜+锡 |

## 7. 军事系统

### 7.1 兵种进化链

```
远古：棍棒兵 → 石斧兵 → 投石兵
      ↓
封建：青铜剑士 → 铁剑士 → 长枪兵（反骑兵）
      战车（→被骑兵淘汰）→ 轻骑兵 → 重骑兵（需马镫）
      猎弓 → 战弓 → 弩手
      投石车/攻城槌
      ↓
现代：火枪兵 → 步枪兵 → 冲锋枪兵（近战火器）
      骑兵 → 龙骑兵（马上火枪）→ 坦克
      加农炮 → 榴弹炮 → 火箭炮
      战斗机/轰炸机
      驱逐舰/巡洋舰/航母
      ↓
未来：机械化步兵/外骨骼战士
      隐形战机/无人机蜂群
      激光防空/电磁炮
      轨道武器（天基打击）
```

### 7.2 兵种克制环

延续 P1.5 的克制环理念，扩展到全时代：

基础克制链条（每时代都适用）：
```
骑兵 ──克制──→ 弓手/远程 ──克制──→ 步兵 ──克制──→ 骑兵
```

特殊克制：
- 长枪兵克制骑兵（2×伤害）
- 攻城器械克制城市/防御工事
- 防空克制空军
- 反隐形卫星克制隐身单位

### 7.3 军队规模约束

不同于 P1.5 的"造多少都行"（仅城市邻格限制），引入：

- **人口池**：每训练一个军事单位，消耗 1 人口（从最近的聚落扣除）。小国养不起大军。
- **粮食维持**：每军事单位每回合消耗 1 粮。和平时期的常备军是财政负担。
- **支持度惩罚**：已有！每战斗单位每回合-1 支持度（穷兵黩武）。
- **动员 vs 常备**：和平时期可以"解散"部分常备军→人口回归聚落+粮食释放。战时再"动员"→短期大量出兵但需支持度+时间。

### 7.4 战争形态演变

| 时代 | 战争特点 | 核心瓶颈 |
|------|---------|---------|
| 远古 | 小规模部落冲突，死几个兵就伤筋动骨 | 人口 |
| 封建 | 贵族精锐+农民征召混编，攻城开始有意义 | 铁+马 |
| 现代 | 大规模动员，火力覆盖，闪电战/阵地战 | 工业产能+石油 |
| 未来 | 精准打击，信息战，太空武器，非对称战争 | 稀土+AI+信息 |

## 8. 诸子百家：封建时代核心决策

这是封建时代最大的决策——永久性选择你的"文明基因"。

### 8.1 可选学派

| 学派 | 核心加成 | 核心代价 | 独特能力 |
|------|---------|---------|---------|
| **法家** | 税收+30%，军队训练速度+50%，法律秩序（叛乱率-80%） | 支持度上限-20，文化产出-50% | **军功授爵**：每消灭一个敌方单位，获得少量天命（以战养战） |
| **儒家** | 支持度上限+30，天命产出+50%，官僚效率+20% | 军事效率-10%，科技偏保守（部分激进科技研究速度-20%） | **以德服人**：可以通过"文化同化"和平获取邻国领土（长期投资） |
| **道家** | 灾难恢复速度+50%，随机科技突破概率+30%，人民自发创新 | 税收-30%，常备军规模上限-30% | **无为而成**：不操作时自动有"隐藏产出"，且自然灾害不扣天命 |
| **墨家** | 防御工事效果+50%，守城攻击力+30%，科技研究不偏科 | 不能主动宣战，外交复杂度增加 | **墨守成规**：可以在任何城市自动获得免费的防御设施；墨家机关→攻城器械效果+25% |
| **兵家** | 军事单位 ATK+15%，战术选项翻倍，战争支持度惩罚减半 | 和平时期产出-10%，外交关系紧张度+30% | **知己知彼**：可以看到邻国军队的确切构成（其他人只能估算） |

### 8.2 诸子百家的交互

- **法家 vs 儒家**：法家抢地快但不能消化（被征服领土支持度低），儒家抢地慢但消化好
- **道家 vs 兵家**：道家靠时间积累优势（越打越有），兵家靠主动出击（不打架就浪费加成）
- **墨家 vs 法家**：墨家是终极"种田流"，法家是终极"征服流"——但如果有墨家邻国，法家打不进去（墨家防守太强），法家的胜利条件需要另想办法

**设计意图**：诸子百家不是简单的 buff/debuff 选择器，而是让你重新定义"什么叫做玩得好"。

## 9. 红白分叉：现代时代核心决策

P1.5 已经在无头引擎上验证了这个机制的核心张力。P2 需要做的是：

1. **保留全部已验证的机制**：支持度系统、组织度积累/兑换、白线周期性危机
2. **扩展红白路线的"叙事力"和"交互深度"**
3. **确保红白和其他系统（诸子百家/聚落/贸易）产生交互效应**

### 9.1 红白路线的深化设计（在 P1.5 基础上）

#### 白线深化：不只是"产出×1.5"

| 子系统 | P1.5 | P2 深化 |
|--------|------|---------|
| 市场机制 | 无 | 白线城市自动产生"市场"建筑——商人阶层出现。商人越富→政治影响力越大→可能推动你走向更极端的市场化（好/坏取决于你的处境） |
| 对外投资 | 无 | 白线可以向落后国家投资建厂→获得持续收益分成。但投资过度→对方经济被你控制→国际紧张 |
| 金融泡沫 | 危机系统已有 | 扩展：不只周期性扣支持度，而是"资产泡沫→破裂→金融危机"的循环。玩家可以干预（提前刺破泡沫 = 小损失，放任 = 可能更大危机） |
| 人才外流 vs 人才流入 | 无 | 白线繁荣→吸引全球人才流入→科研加速。但如果危机爆发→人才大量外流到红线国家 |

#### 红线深化：不只是"组织度→爆发"

| 子系统 | P1.5 | P2 深化 |
|--------|------|---------|
| 根据地 | 无 | 红线玩家可以在农村格建立"根据地"——村子产出天命+支持度，被攻击时自动生成民兵（不占人口池） |
| 土改 | 抽象扩张已有 | 深化：红线可以进行"土地改革"事件→一次性大幅提升支持度+农村产出，但触发地主阶级反弹（城市暂时动荡） |
| 国际主义 | 无 | 红线可以向其他国家的"受压迫阶层"输出革命→在该国制造内部动荡（非军事攻击手段） |
| 五年计划 | 无 | 红线可以设定 N 回合的"五年计划"→选定一个领域（重工业/科技/军事）→持续投入→N回合后获得巨大收益，但中间不能切换 |
| 自力更生 | 无 | 红线被全面封锁时（贸易禁运），自动触发"自力更生"——所有被禁运的资源可以花额外时间自产替代品 |

#### 红白交互：不只是克制

| 交互 | 机制 |
|------|------|
| 白→红 | 白线繁荣→商品倾销到红国家→红国本土产业受损（但物价便宜） |
| 红→白 | 红国在落后国家输出革命→白国在那里的投资被国有化→白国损失 |
| 代理人战争 | 红白不在自己领土上打，而是资助小国内战（类似朝鲜战争/越南战争的逻辑） |
| 科技竞赛 | 红白在关键科技（核/太空/AI）上形成"竞赛"——谁先到谁获得巨大优势 |

## 10. 天命系统

### 10.1 天命是什么

天命 = "政权合法性的综合评分"，范围 0-100。

不同于文明六的"忠诚度"（只影响城市叛变）和 P1.5 的"支持度"（只影响产出），天命是**更高维度**的指标：

- 天命高（80-100）：**繁荣时代** → 所有产出有加成，人民自发参军（免费单位），邻国向你靠拢
- 天命中（40-80）：**正常状态** → 无特殊效果
- 天命低（20-40）：**衰败时代** → 产出惩罚，可能爆发农民起义（内部敌人出现！）  
- 天命极低（0-20）：**天命已失** → 你的城市可能直接叛变到邻国（尤其如果对方天命高），大规模起义

### 10.2 影响天命的行为

| 行为 | 天命变化 | 说明 |
|------|---------|------|
| 完成超级工程 | +10~20 | 一次性的巨大威望 |
| 军事胜利（胜仗） | +3~5/场 | 打败强敌更多 |
| 军事失败（败仗） | -5~10/场 | 割地赔款更惨 |
| 饥荒/瘟疫/自然灾害 | -5~15 | 被认为"天降惩罚" |
| 大兴土木（过度） | -2/回合 | 劳民伤财 |
| 完成科技创新 | +1~3 | 首次完成（别人做过的没有加成） |
| 举行祭天/封禅 | +5（花费大量资源） | "买"天命 |
| 镇压起义 | -3（短期） | 镇压会增加恐惧但减少民心 |
| 轻徭薄赋 | +1/回合 | 持续积累 |
| 苛捐杂税 | -1/回合 | 持续消耗 |
| 屠杀/暴政 | -20 | 一次性巨大惩罚 |
| 与邻国友好 | +0.5/回合/国 | 睦邻 |
| 被侵略 | +5（"保家卫国"加成） | 防御战反而涨天命 |

### 10.3 天命的跨时代演变

| 时代 | 天命来源变化 |
|------|------------|
| 远古 | 主要来自祭祀/祖先崇拜，对自然灾害的应对 |
| 封建 | 主要来自诸子百家的实践、王道（儒家）/霸道（法家）的选择 |
| 现代 | 主要来自经济表现（GDP）、民族主义、意识形态吸引力 |
| 未来 | 主要来自科技领先（谁先殖民火星？）、人类命运的引领者角色 |

## 11. 迷雾与信息战

### 11.1 FOW 三层状态

| 状态 | 含义 | 能看到什么 |
|------|------|----------|
| **可见** | 当前有己方单位或城市视野覆盖 | 完全信息：地形、敌军、资源 |
| **已探索** | 曾经探索过但当前无视野 | 地形（不更新）、设施（不更新）、不知道当前有谁在 |
| **迷雾** | 从未探索过 | 完全不可见 |

### 11.2 视野范围

| 单位/建筑 | 视野（格数） | 说明 |
|----------|------------|------|
| 侦察兵 | 3 | 远古核心探图单位 |
| 步兵/骑兵 | 1 | 基本视野 |
| 弓手 | 2 | 需要视野来射击 |
| 瞭望塔（建筑） | 4 | 需建造，提供固定视野 |
| 城市 | 2 | 初始，可升级 |
| 骑兵巡逻 | 2 | 骑兵在己方领土内额外+1 |
| 烽火台（边境建筑） | 5 | 只能看到"有敌军入境"（不显示具体兵种数量） |
| 热气球（现代） | 6 | 静态高视野 |
| 侦察机（现代） | 直线 10 格 | 飞一行，沿路照亮 |
| 卫星（未来） | 全球低分辨率 | 所有格变为"已探索"（但不能实时看到敌军移动） |

### 11.3 信息不对称的战术价值

- **伏击**：己方军队在森林/山地格 → 对邻格敌人不可见（除非有侦察单位在旁）。被伏击的第一回合防御方 ATK+5
- **佯攻**：在可见边界制造"假大军"（耗费少量资源）→ 对方以为你大军压境→消耗对方资源防御错误方向
- **信息战（未来）**：可以黑入对方卫星/电报→窃取视野。也可以伪造信息（让对方看到不存在的军队）

## 12. 外交与贸易体系

### 12.1 外交关系

每个 AI/人类玩家之间的外交关系是一个连续值（-100 到 +100），受以下影响：

| 因素 | 影响方向 |
|------|---------|
| 贸易量 | + |
| 共同敌人 | + |
| 边界接壤 | -（领土争端） |
| 意识形态冲突（红 vs 白） | - |
| 相同诸子百家 | + |
| 背刺/违约 | 巨大 - |
| 无偿援助 | + |
| 军事力量对比 | 弱对强 +（畏惧），强对弱 -（轻视）|
| 天命差距 | 高天命吸引低天命邻国 |

### 12.2 外交选项

| 选项 | 条件 | 效果 |
|------|------|------|
| 贸易协定 | 关系≥0 | 双方商路收入+20% |
| 同盟（防御） | 关系≥50 | 一方被攻击→另一方自动参战 |
| 军事同盟（进攻+防御） | 关系≥70 | 可以同步宣战、协调进攻 |
| 朝贡关系 | 力量差距大 | 弱方定期纳贡→强方保护 |
| 和平条约 | 战后 | N 回合内不能开战 |
| 割地/赔款 | 战后 | 战败惩罚 |
| 联姻 | 双方都是封建 | 关系永久+20 |
| 意识形态输出 | 已有对应制度 | 让对方逐渐偏向己方路线 |
| 经济制裁 | 关系差 | 对方贸易收入-30% |
| 谴责 | 任何 | 关系-10，对方天命轻微下降 |

### 12.3 贸易依赖度（保留 P2 讨论的高质量设计）

你在之前的讨论中已经设计了一个非常好的依赖度系统——保留它：

- 每回合接受贸易 → 依赖度+1
- 断供 → 戒断反应（产出惩罚持续到依赖度归零）
- 自主替代 → 花资源加速脱钩
- 完全自主后可自产

**P2 深化**：
- 依赖度不仅影响经济，也影响外交——依存方在外交上必须对供应方做出让步（"软殖民"）
- 多国同时断供→"国际封锁"效果叠加（红线会触发"自力更生"补偿机制）

## 13. 超级工程与国家项目

### 13.1 超级工程

超级工程 = 跨越多个回合、消耗大量资源、建成后永久改变游戏状态的巨构。

| 工程 | 时代 | 花费 | 建成年限 | 效果 |
|------|------|------|---------|------|
| **长城**（分段） | 封建 | 每段 50 木 20 金 | 5T/段 | 该段边境格游牧民族无法进入 |
| **大运河** | 封建→现代 | 200 木 100 金 | 15T | 连通两城→贸易收入×2，军队沿河移动速度×2 |
| **都江堰** | 封建 | 80 木 50 金 | 10T | 半径 5 格内所有农场粮产+2（永久） |
| **紫禁城** | 封建→现代 | 200 金 50 木 | 12T | 首都天命+15，文化产出×2 |
| **工业革命**（国家项目） | 现代 | 400 产能 | 20T | 全国工厂产出+100%，污染开始 |
| **曼哈顿计划**（原子弹） | 现代 | 500 科技 200 产能 | 15T | 解锁核弹制造 |
| **阿波罗计划** | 未来 | 800 科技 300 产能 | 18T | 解锁月球基地 |
| **AI 觉醒** | 未来 | 1000 科技 | 25T | 解锁终极 AI——效果因你的"AI对齐"制度而异 |
| **戴森云** | 未来 | 5000 产能 | 30T | 终极能源（无限能源，胜利条件之一） |

### 13.2 国家项目（非地图工程）

不同于超级工程（在地图上有物理存在/永久改造地图），国家项目是"全国性的政策投资"：

| 项目 | 效果 |
|------|------|
| 科举改革 | 官员质量提升（减少腐败/管理成本） |
| 土地改革 | 支持度大幅提升，生产效率短期降后升 |
| 五年计划（红线） | 选一领域→N回合集中建设→完成时该领域产出暴增 |
| 马歇尔计划（白线） | 花费资源→友好国家经济增长→你的贸易收入也随之增加 |
| 全民皆兵 | 人口→军队动员率暴增（适合防御战） |

## 14. 随机事件与动态叙事

确定性引擎 ≠ 没有叙事。用 seed 决定的事件序列可以既有"随机感"又有"可复现性"。

### 14.1 事件类型

| 类别 | 例子 | 影响 |
|------|------|------|
| 自然灾害 | 黄河改道、地震、蝗灾、瘟疫 | 多格物资产出骤降、人口损失、天命- |
| 天象 | 彗星/日食 | 天命小波动（远古时代影响大，现代影响小） |
| 科技突破 | 某学者灵光一现 | 当前研究科技加速 |
| 起义/叛乱 | 某地流民起事 | 地图上出现"叛军单位" |
| 外交事件 | 邻国新君即位、商队被劫 | 关系变化/贸易中断 |
| 伟人出现 | 孙子/张衡/诸葛亮型人物 | 一次性效果或持续 buff |
| 文化事件 | 诗歌/戏剧/哲学流派兴起 | 天命+文化产出 |

### 14.2 事件的"代理性"

不是所有事件对所有人都一样：
- 法家对"起义"有天然压制（叛乱概率更低）
- 道家对"自然灾害"有天然抗性（天命不扣）
- 墨家对"科技突破"有触发概率加成

## 15. 胜利条件

### 15.1 多种胜利路径

| 胜利路径 | 条件 | 对应路线 |
|---------|------|---------|
| **征服胜利** | 消灭所有对手（或成为唯一拥有城市者） | 法家/兵家 |
| **科技胜利** | 完成终极科技（如核聚变+太空殖民+AI觉醒三者完成其二）并保持领先 N 回合 | 墨家 |
| **文化胜利** | 天命达到 90 并持续 20 回合（所有对手都认可你为"天命所归"） | 儒家/道家 |
| **经济胜利** | 财富总量超过所有对手之和（垄断全球经济） | 白线 |
| **人民胜利** | 红线专属：所有对手的支持度低于 20 而你高于 80（全球革命） | 红线 |
| **时间胜利** | 500T 结束时得分最高（兜底，正常对局不该触发） | 任何人 |

### 15.2 胜利条件的设计意图

- 不同路线有不同的"舒适胜利路径"，但不是锁死的——法家也可以走科技胜利（秦的标准化+效率）
- 胜利条件之间有**互斥张力**：走征服 → 天命难高（穷兵黩武扣天命），走文化 → 军队不能太强（维持军费消耗资源）
- **同一局里，不同玩家追求的胜利条件应该不同**——这是策略深度的根本来源

## 16. AI 路线图

### 16.1 AI 分层策略

P2 需要保证每个新系统至少有一个"不会犯傻"的 AI 能玩。策略：

| 层级 | AI | 职责 |
|------|-----|------|
| **裁判层** | FlatMC → 升级为真 MCTS | 深度验证、机制评估、参数扫描 |
| **学习层** | Evo NN / BC 重做 / RL | 自我进化、探索策略上界 |
| **基线层** | 手写规则 AI（Greedy/StateAware 升级版） | 性能基线、快速测试 |
| **内容层** | 特定探针（Builder/Rusher 等） | 机制存活判据 |

### 16.2 FlatMC → 真 MCTS 升级

当前 FlatMC 是 1-ply minimax + rollout。P2 需要升级为：

- **多步搜索**：depth ≥ 4（取代当前 1-ply）
- **UCB1 选择**：替代 minimax（更适合部分可观测 + 大动作空间）
- **FOW-aware rollout**：rollout 中使用"真实可见信息"而非全局信息
- **对手模型多样化**：对手池包含多种手写 AI（修复 P1.5 发现的对手过拟合问题）

### 16.3 NN 训练升级

- 当前 NN：2000 参数，Adam，最高 0.863
- P2 目标：5000-10000 参数，PPO + 自对弈，目标胜率 > 70%
- 关键改进：特征工程覆盖新系统（聚落数/天命/红白状态等），让 NN 能"感知"到新机制的维度

### 16.4 BC 重做：整体 Turn-Plan 预测

P1.5 的 BC 失败是因为"每个动作独立预测 → 动作不协调"。P2 重做方向：

- 使用序列模型（LSTM 或小型 Transformer）
- 输入：当前游戏状态
- 输出：整个回合的动作序列（一个 complete turn-plan）
- 训练数据：FlatMC 高质量自对弈数据

### 16.5 RL（PPO/DQN）探索

- 在 P2 中后期（机制基本稳定后）尝试 RL
- 目标：验证"学习型 AI 能否在 P2 引擎上超越手写规则和 FlatMC"
- 如果 RL 能突破 → 游戏深度真实存在
- 如果 RL 不能突破而 FlatMC 深度曲线也无平台期 → "AI 还不够好"（不是游戏不够深）

### 16.6 AI 适配检查清单

每个新系统实现后，跑一次完整矩阵（FlatMC×手写AI×探针），确保：
- FlatMC 胜率 > 50%（否则新系统太复杂，AI 盲打）
- 至少 1 个手写 AI 能利用新系统（否则新系统可能是"装饰"）
- 不同 AI 在新系统下的克制关系不消失（否则新系统杀死了原有的博弈）

## 19. 风险与判断

### 19.1 核心风险

| 风险 | 严重度 | 缓解措施 |
|------|--------|---------|
| 系统太多，AI 无法全部适配 | 🔴 高 | 每个阶段确保 FlatMC 能玩再推进 |
| 诸子百家+红白+天命+制度树 交织→爆炸级的平衡问题 | 🔴 高 | 每步跑矩阵，不能全做完再调 |
| WASM 编译有性能问题 | 🟡 中 | 阶段 4 前做 POC |
| 没人玩→没有人类反馈→回归循环论证 | 🟡 中 | 最少找 3 个朋友 |
| FOW 做完发现 AI 无法处理部分可观测性 | 🟡 中 | FlatMC 先做 FOW-aware，手写 AI 允许退化 |
| 开发时间太长→热情耗尽 | 🟡 中 | 每个阶段有"可玩产物"（不只是矩阵数据） |

### 19.2 哲学风险

- **"中国历史模拟"还是"以中国历史为灵感的好游戏"？** —— 倾向后者。史记说"微管仲，吾其被发左衽矣" → 好的机制不用非要"像"历史。大运河在游戏里的价值是"连通两城→贸易翻倍"，不是"隋炀帝劳民伤财"。素材库里的历史是为机制找灵感，不是机制为历史找说教。

- **"深度"的评判标准仍然模糊** —— 延续 P1.5 的深度规格法：每个新机制必须预先写"怎么算它成立"，用 `depth` 工具验证。如果无法验证（机制太软/无 AI 能用），先搁置不做。

- **红白线的"中立/灰线"是否破坏二分张力？** —— 灰线的意义在于"有人不想选极端"——但灰线当然应该难赢。因为极端才有戏剧性。灰线是"难度最高"的路线。
