# S2 立裁判 — 裁决(FlatMC 非循环裁判)

> 2026-07-11 | 第四个 AI | 复现: `cargo run --release --bin judge -- 60 0 20`
> 前置: 默认配置已=甜点(成本×2 HP160), 见 config.rs / SCAN-FINDINGS.md。

## 为什么造 FlatMC(破循环)

旧 `Search`/`depth` 只在 4 个手写剧本里选、rollout 也用这 4 个 → "无支配/有深度"退化成
"我的 5 个脚本互相不碾压"(循环论证)。**FlatMC 操作层级在剧本之下**: 每回合从真实动作基元
菜单(研究 ∈ 所有可研究科技 / 生产 ∈ 所有兵种含侦察兵 / 姿态 ∈ 进攻·防守·骚扰·按兵)组合,
能表达剧本表达不了的行为。minimax over {建设/步兵rush/骑兵rush} 三种对手响应, 全 rollout 打到底。

## 裁决矩阵(FlatMC vs 手写剧本, 60 seeds paired, 默认甜点)

| 对手 | FlatMC 胜率 | 均结束T |
|------|-----------|---------|
| Builder | 49.2% | 39.0 |
| Rusher | 45.0% | 13.2 |
| CavRusher | 50.0% | 21.2 |
| Defender | 47.5% | 40.7 |
| Adaptive | 48.3% | 38.9 |
| **平均** | **≈48%** | — |

**双向都无支配**: 没有任何剧本碾压 FlatMC(最差 Rusher 也只 55:45), FlatMC 也碾压不了任何剧本
(最好 ~50%)。**甜点带的策略空间是平的。**

## 死机制使用率(40 局, FlatMC 自发?)

- 生产: worker 33, archer 5, cavalry 2, scout 2, infantry 1 —— **侦察兵/弓箭手/骑兵都被摸到**(非零),
  但极少(主要在建设经济)。
- 研究: 22% 偏离 C 主链(研究了 M/E/C2)。

FlatMC **不像固定剧本那样无视死机制**(它会产侦察兵/弓箭手、走非标准科技顺序), 但**这些多样性没换来任何胜率**。

## 裁决: 偏 H0(深度温和/偏浅), 但比旧结论硬

**这是项目第一次用"非循环"工具量深度。** 结论:

1. **无隐藏支配漏洞**: 一个能跳出剧本、能搜索的 AI 在甜点带找不到碾压策略 → 甜点没有被某个
   固定策略统治的洞(好, 印证甜点健康)。
2. **应变/机制丰富度买不到边际**: FlatMC 能用侦察兵/弓箭手/非标准科技, 却只打平剧本 →
   **简单策略接近最优 → 深度温和、偏浅**。死机制不是"等待解锁的潜在深度", 是"当前设计不奖励"。
3. **推论**: 真正的深度增加需要**机制/内容改动**(让不同选择有不同回报), 而非精调参数。
   这印证并**收紧**了旧"一阶深度温和成立"——现在有非循环证据, 而非弱探针互测。

## 诚实边界(触发下一阶段的残留)

- FlatMC 平均 48%, **略低于剧本而非高于**。所以不能完全区分"游戏浅"和"我的裁判还不够强"。
- FlatMC 是 1-ply + rollout, 天花板 = 默认策略 + 一步搜索; minimax 只 3 种对手假设。
- **能确证的**: 甜点无支配洞 + 死机制无回报 + 简单策略近最优 → 深度偏浅是**有力信号**。
- **不能确证的**: 一个更强的 AI(真 MCTS / 学习型)是否会挖出 FlatMC 够不到的深度。
  → 这正是止损判据里预留的下一阶段(见 stage-S2-goal-acceptance.md)。

## 方法论收获(新)

- **"机制没人用"现在是可测量的**: 让一个能跳出剧本的 AI 去玩, 数它用不用死机制 + 死机制换不换胜率。
  用了但不换胜率 = 机制在当前设计下无回报(该改设计或该砍), 不是 AI 不会用。
- **裁判的价值是诚实裁决, 不是刷胜率**: 中途一度想把 FlatMC 调到碾压剧本, 及时刹住——那正是
  项目掉过的"调出新 speedrun"坑。裁判停在"无盲点的平手"就够裁决了。

## 性能债(发现, 待办)

评估/裁判全仓库**纯单线程**(无 rayon, seed 循环串行), FlatMC 矩阵跑几分钟只用 1 核。
paired seeds 天然独立, `par_iter` 可近线性加速(N=核数)。规范里也无"性能"纪律。
→ 建议: 加 rayon 并行 eval/judge 外层 seed 循环(下一步, 低风险高收益)。
