## 学习笔记格式

每个 Phase 完成后，创建 `phase-N-<名称>.md`：

```markdown
# Phase N: <名称> — 学习笔记

## Rust 概念
- 新学的概念（用自己的话解释）
- 和 Python 的对比（哪些地方像，哪些地方不像）

## 游戏设计理解
- 这个模块解决了什么游戏设计问题？
- 有没有之前没想到的设计考量？

## 模糊点
- 哪里还不太理解？
- 想进一步了解什么？

## 一行总结
一句话总结这个 Phase 的核心收获。
```
